# Лекция 2. Операции над случайными событиями

## 2.1. Соотношения между случайными событиями

Любую совокупность $A=\{\omega_{1}, \omega_{2}, ..., \omega_{k}\}$ из описаний элементарных исходов называем случайным событием (определение 2, лекция 1).

Понятие случайного события имеет абстрактный характер, так как конкретная природа возникновения события не имеет значения. Поэтому между случайными событиями могут существовать соотношения только логического и теоретико-множественного характера.

**Определение 1.** Если каждое описание $\omega$ случайного события $A$ принадлежит случайному событию $B$, то этот факт отношения между $A$ и $B$ обозначается $A \subset B$. В этом случае говорят, что $A$ есть часть $B$ или $B$ включает $A$. Из отношения $A \subset B$ вытекает, что при каждом осуществлении комплекса условий $\Sigma$ при проведении эксперимента $E$ вместе с исходом $A$ обязательно наступает исход $B$ (событие $A$ влечет $B$).

**Определение 2.** Если $A \subset B$ и $B \subset A$, то мы будем говорить, что события $A$ и $B$ равносильны, и обозначать это через $A = B$.

**Определение 3.** Два события $A$ и $B$ называются несовместными, если они не содержат общих описаний элементарных исходов. В противном случае $A$ и $B$ называются совместными, и тогда они содержат хотя бы один общий элемент $\omega \in \Omega$. Если события $A$ и $B$ являются несовместными, то наступление одного из них исключает появление другого.

**Определение 4.** Событие, которое содержит все такие описания $\omega \in \Omega$ элементарных исходов, которые не принадлежат некоторому событию $A$, называется противоположным по отношению к событию $A$ и обозначается $\overline{A}$. Если события $A$ и $\overline{A}$ являются противоположными, то событие $A$ происходит тогда и только тогда, когда не происходит событие $\overline{A}$.

> **Отметим:** $\overline{\overline{A}} = A$, $\overline{\Omega} = \emptyset$, $\overline{\emptyset} = \Omega$.

### Пример 1

Рассмотрим следующий эксперимент. На поверхность стола наудачу один раз подбрасывается игральная кость. Грани кости занумерованы от 1 до 6. В этом опыте комплекс условий имеет вид:

$\Sigma = \{u_{1}$ — игральная кость, $u_{2}$ — заданный механизм подбрасывания, $u_{3}$ — одноразовое подбрасывание кости, $u_{4}$ — характеристики поверхности стола, $u_{5}$ — достаточная освещённость поверхности стола для фиксации числа очков на выпавшей грани, ..., случайные факторы $s=5$: температура, давление, колебание воздуха и т. п.$\}$

Пусть при $i = 1, 2, ..., 6$ элементарный исход $A^{\prime}_{i}$ означает выпадение грани с номером $i$. Тогда $\mathfrak{J}^{\prime} = \{A^{\prime}_{1}, A^{\prime}_{2}, ..., A^{\prime}_{6}\}$.

Будем описывать (кодировать) элементарный исход $A^{\prime}_{i}$ символом $\omega_{i}$. Тогда $\omega_{i}$ — элементарное случайное событие и $\Omega = \{\omega_{1}, \omega_{2}, \omega_{3}, \omega_{4}, \omega_{5}, \omega_{6}\}$.

Рассмотрим следующие исходы эксперимента:

* $B$ — {выпадение четной грани с номером не больше пяти}
* $C$ — {появление четной грани}
* $D$ — {появление нечетной грани}
* $H$ — {выпадение четной грани, номер которой не делится на шесть}

Через элементарные события данные исходы выражаются следующим образом:
$B = \{\omega_{2}, \omega_{4}\}$
$C = \{\omega_{2}, \omega_{4}, \omega_{6}\}$
$D = \{\omega_{1}, \omega_{3}, \omega_{5}\}$
$H = \{\omega_{2}, \omega_{4}\}$

Отсюда получаем следующие соотношения: $B \subset C$, $B = H$ и $D = \overline{C}$.

## 2.2. Теоретико-множественные операции над случайными событиями

**Определение 5.** Объединением случайных событий $A$ и $B$, обозначаемым через $A \cup B$, называется такое случайное событие, которое содержит все описания события $A$ и все те описания события $B$, которые не входят в $A$.

> **Отметим:** $A \cup A = A$, $A \cup \overline{A} = \Omega$, $A \cup \Omega = \Omega$, $A \cup \emptyset = A$.
> Если $A \subset B$, то $A \cup B = B$.

Если событие $A_{i}$ задано при каждом $i = 1, 2, ...$, то их объединение есть такое случайное событие $C = \bigcup_{i=1}^{\infty} A_{i}$, каждое описание которого принадлежит хотя бы одному из событий $A_{i}$. В терминах наступления случайных событий операцию объединения можно проинтерпретировать следующим образом: если $C = A_{1} \cup A_{2} \cup ...$, то случайное событие $C$ происходит тогда и только тогда, когда происходит, по крайней мере, одно из случайных событий $A_{1}, A_{2}, ...$.

**Определение 6.** Пересечением двух случайных событий $A$ и $B$, обозначаемым $A \cap B$, называется такое случайное событие, которое содержит только те описания, которые принадлежат как $A$, так и $B$.

> **Отметим:** $A \cap A = A$, $A \cap \overline{A} = \emptyset$, $A \cap \emptyset = \emptyset$, $A \cap \Omega = A$.
> Если $A \subset B$, то $A \cap B = A$.

Если событие $A_{i}$ задано при каждом $i = 1, 2, ...$, то их пересечение есть такое случайное событие $D = \bigcap_{i=1}^{\infty} A_{i}$, каждое описание которого принадлежит одновременно всем случайным событиям $A_{1}, A_{2}, ...$. В терминах наступления случайных событий равенство $D = A_{1} \cap A_{2} \cap ...$ означает, что событие $D$ происходит тогда и только тогда, когда происходят все случайные события $A_{1}, A_{2}, ...$ одновременно.

**Определение 7.** Разностью случайных событий $A$ и $B$ называется такое случайное событие, которое содержит описания элементарных исходов из $A$ и не содержит описания элементарных исходов из $B$. Разность событий $A$ и $B$ обозначается $A \setminus B$. Если $C = A \setminus B = A \cap \overline{B}$, то случайное событие $C$ происходит, когда событие $A$ произошло, а $B$ не произошло.

**Определение 8.** Симметрической разностью случайных событий $A$ и $B$, которая обозначается символом $A \triangle B$, называется случайное событие следующего вида: $A \triangle B = (A \setminus B) \cup (B \setminus A)$. Если $C = A \triangle B$, то случайное событие $C$ происходит тогда, когда наблюдается только одно из событий — либо $A$, либо $B$.

**Определение 9.** События $A_{1}, A_{2}, ...$ образуют полную группу событий, если их объединение равно достоверному событию $\Omega$.

**Определение 10.** События $A_{1}, A_{2}, ...$ образуют полную группу несовместимых событий, если их объединение равно достоверному событию $\Omega$, и они попарно являются несовместными. События $A$ и $\overline{A}$ образуют полную группу несовместных событий, так как $A \cup \overline{A} = \Omega$ и $A \cap \overline{A} = \emptyset$.

### Пример 3

На прямоугольный участок земли за некоторый фиксированный промежуток времени из космоса падает частица. Достоверному исходу, который означает попадание частицы на прямоугольный участок, можно поставить в соответствие событие $\Omega = \{\omega = (x,y) : 0 \le x \le a, 0 \le y \le b\}$. Произвольное элементарное событие $A^{\prime} = \{\omega = (x,y)\}$ означает прохождение частицы через точку с абсциссой $x$ и ординатой $y$.

Определим исходы $A, B, C, D$ следующим образом:

* $B$ — {попадание частицы в эллипс, большая полуось которого расположена горизонтально}

## 2.3. Основные законы теоретико-множественных операций

Теоретико-множественные операции над произвольными случайными событиями удовлетворяют следующим основным законам:

**1. Переместительный (коммутативный) закон:**


$$A \cup B = B \cup A, \quad A \cap B = B \cap A$$

**2. Сочетательный (ассоциативный) закон:**


$$A \cup B \cup C = (A \cup B) \cup C = A \cup (B \cup C)$$

$$A \cap B \cap C = (A \cap B) \cap C = A \cap (B \cap C)$$

**3. Распределительный (дистрибутивный) закон:**


$$(A \cup B) \cap C = (A \cap C) \cup (B \cap C)$$

$$(A \cap B) \cup C = (A \cup C) \cap (B \cup C)$$

**4. Закон де Моргана:**


$$\overline{A \cup B} = \overline{A} \cap \overline{B}$$

$$\overline{A \cap B} = \overline{A} \cup \overline{B}$$

### Пример 4

Предположим, что $A, B, C$ — три заданных случайных события. Выразим формулировки указанных ниже событий через заданные:

1. Произошло только событие $A$: $A \cap \overline{B} \cap \overline{C} = A \setminus (B \cup C)$
2. Произошли события $A$ и $B$, $C$ не произошло: $A \cap B \cap \overline{C} = (A \cap B) \setminus C$
3. Все три события произошли: $A \cap B \cap C$
4. Произошло по крайней мере одно из трех событий: $A \cup B \cup C$
5. Произошло одно и только одно событие из трех: $(A \cap \overline{B} \cap \overline{C}) \cup (\overline{A} \cap B \cap \overline{C}) \cup (\overline{A} \cap \overline{B} \cap C)$
6. Ни одного события не произошло: $\overline{A \cup B \cup C} = \overline{A} \cap \overline{B} \cap \overline{C}$
7. Произошло не более двух событий: $\overline{A \cap B \cap C} = \overline{A} \cup \overline{B} \cup \overline{C}$
8. Произошло по крайней мере два события: $(A \cap B) \cup (B \cap C) \cup (A \cap C)$

### Пример 5

Относительно перечисленных ниже событий некоторого эксперимента указать, образуют ли они полную группу.

**1. Осуществляется передача двух сигналов ($E$).**
Рассматриваются события:
$D_{1}$ — {хотя бы один сигнал не искажен}
$D_{2}$ — {хотя бы один сигнал искажен}

Для данного эксперимента множество описаний элементарных исходов имеет следующий вид: $\Omega = \{\omega_{1} = (1,1), \omega_{2} = (1,0), \omega_{3} = (0,1), \omega_{4} = (0,0)\}$, где «1» — сигнал искажен, «0» — сигнал не искажен.
Получаем выражения для событий через элементы $\Omega$:
$D_{1} = \{\omega_{2}, \omega_{3}, \omega_{4}\}$
$D_{2} = \{\omega_{1}, \omega_{2}, \omega_{3}\}$
Следовательно, $D_{1} \cup D_{2} = \{\omega_{1}, \omega_{2}, \omega_{3}, \omega_{4}\} = \Omega$.
Значит, события $D_{1}$ и $D_{2}$ **образуют** полную группу.

**2. Осуществляется передача трех сообщений ($E$).**
Рассматриваются события:
$C_{1}$ — {в первом сообщении есть ошибка}
$C_{2}$ — {во втором сообщении есть ошибка}
$C_{3}$ — {в третьем сообщении есть ошибка}

Здесь $\Omega = \{\omega_{1} = (0,0,0), \omega_{2} = (0,0,1), \omega_{3} = (0,1,0), \omega_{4} = (1,0,0), \omega_{5} = (0,1,1), \omega_{6} = (1,0,1), \omega_{7} = (1,1,0), \omega_{8} = (1,1,1)\}$, где «1» — в сообщении есть ошибка, «0» — ошибки нет.
$C_{1} = \{\omega_{4}, \omega_{6}, \omega_{7}, \omega_{8}\}$
$C_{2} = \{\omega_{3}, \omega_{5}, \omega_{7}, \omega_{8}\}$
$C_{3} = \{\omega_{2}, \omega_{5}, \omega_{6}, \omega_{8}\}$

Получаем, что $C_{1} \cup C_{2} \cup C_{3} = \{\omega_{2}, \omega_{3}, \omega_{4}, \omega_{5}, \omega_{6}, \omega_{7}, \omega_{8}\} \neq \Omega$ (не хватает $\omega_{1}$).
Значит, события $C_{1}, C_{2}$ и $C_{3}$ **не образуют** полную группу.

## Заключение

1. Между случайными событиями могут существовать соотношения только логического и теоретико-множественного характера.
2. Выделяют следующие теоретико-множественные операции над случайными событиями: объединение, пересечение, разность, симметрическая разность.
3. Существует четыре основных закона для теоретико-множественных операций над случайными событиями: коммутативный, ассоциативный, дистрибутивный и закон де Моргана.
4. Для выяснения соотношений между случайными событиями и при проведении операций над ними, следует опираться на определения событий через описания элементарных исходов, а не на словесные формулировки.
