# Лекция 12. Второстепенные числовые характеристики одномерных случайных величин

## 12.1. Начальные и центральные моменты к-го порядка

**Второстепенные числовые характеристики** одномерной случайной величины — это числовые константы, отражающие специфические свойства её законов распределения. К ним относятся начальные и центральные моменты, квантили, медиана, мода, коэффициент асимметрии и эксцесс.

На практике часто вводят характеристики, которые учитывают маловероятные, но большие по абсолютной величине значения случайной величины. Эти характеристики называют **начальными и центральными моментами**.

**Определение 1.** **Начальным моментом к-го порядка** случайной величины называется математическое ожидание к-ой степени случайной величины:


$$\alpha_{k}(\xi) = M(\xi^k), \quad k = 0, 1, ...$$

* При $k=0$: $\alpha_{0}(\xi) = M(1) = 1$.
* При $k=1$: $\alpha_{1}(\xi) = M(\xi)$ (математическое ожидание).

**Формулы для вычисления:**

* В общем случае: $\alpha_{k}(\xi) = \int_{-\infty}^{+\infty} x^k dF(x)$
* Для дискретной случайной величины: $\alpha_{k}(\xi) = \sum_{i=1}^{\infty} (x_i)^k p_i$
* Для непрерывной случайной величины: $\alpha_{k}(\xi) = \int_{-\infty}^{+\infty} x^k f(x) \, dx$

**Важно!** Начальные моменты $\alpha_k(\xi)$ учитывают большие по абсолютной величине и маловероятные значения случайной величины за счёт возведения в $k$-ю степень (часто говорят, что они учитывают «хвосты» распределения).

**Определение 2.** **Центральным моментом к-го порядка** случайной величины $\xi$ (при $M|\xi| < \infty$) называется математическое ожидание $k$-ой степени отклонения случайной величины от её математического ожидания:


$$\beta_{k}(\xi) = M(\xi - M\xi)^k, \quad k = 0, 1, ...$$

* При $k=0$: $\beta_{0}(\xi) = 1$.
* При $k=1$: $\beta_{1}(\xi) = 0$.
* При $k=2$: $\beta_{2}(\xi) = D(\xi)$ (дисперсия).

**Формулы для вычисления:**

* В общем случае: $\beta_{k}(\xi) = \int_{-\infty}^{+\infty} (x - M\xi)^k dF(x)$
* Для дискретной случайной величины: $\beta_{k}(\xi) = \sum_{i=1}^{\infty} (x_i - M\xi)^k p_i$
* Для непрерывной случайной величины: $\beta_{k}(\xi) = \int_{-\infty}^{+\infty} (x - M\xi)^k f(x) \, dx$

## 12.2. Квантиль порядка $p$, медиана и мода

Если математическое ожидание не существует, рассматривают другие характеристики положения.

**Определение 3.** **Квантиль порядка $p$** $(0 < p < 1)$ случайной величины $\xi$ — это такое число $K_p\xi$, для которого выполняется соотношение:


$$F(K_p\xi) \le p \le F(K_p\xi + 0)$$


Условия можно записать как: $P(\xi < K_p\xi) \le p$ и $P(\xi > K_p\xi) \le 1 - p$.
Квантили существуют для любой случайной величины.

**Определение 4.** **Медиана** случайной величины $\xi$ — это квантиль порядка $p=0,5$:


$$Me(\xi) = K_{1/2}\xi$$


Медиана удовлетворяет неравенствам: $P(\xi < Me\xi) \le 1/2$ и $P(\xi > Me\xi) \le 1/2$.
Для непрерывной случайной величины медиана находится из уравнения $F(Me\xi) = 1/2$.

**Определение 5.** **Модой** ($Mo\xi$) дискретной случайной величины называется такое её возможное значение $x_i$, что $p_{i-1} \le p_i$ и $p_i \ge p_{i+1}$.

**Определение 6.** **Модой** ($Mo\xi$) непрерывной случайной величины называется такое значение $x_{max}$, которое обеспечивает плотности распределения $f(x)$ локальный максимум.

* Если мода единственна, распределение называется *унимадальным*, в остальных случаях — *мультимодальным*.

**Определение 7 и 8 (Наивероятнейшее значение).** * Для дискретной: $Mo^*\xi = x_r$, где $p_r = \max\{p_k\}$.

* Для непрерывной: $Mo^*\xi$ обеспечивает $f(Mo^*\xi) = \max f(x)$.

## 12.3. Коэффициент асимметрии, эксцесс

Эти характеристики отвечают за визуальную форму распределения.

**Определение 9.** **Коэффициент асимметрии** случайной величины:


$$K_a(\xi) = \frac{\beta_3(\xi)}{(\sigma(\xi))^3}$$

* Если $K_a(\xi) = 0$, распределение симметрично.
* Если $K_a(\xi) < 0$ — отрицательная асимметрия.
* Если $K_a(\xi) > 0$ — положительная асимметрия.

**Определение 10.** **Эксцесс** случайной величины:


$$\partial(\xi) = \frac{\beta_4(\xi)}{(\sigma(\xi))^4} - 3$$


Эксцесс характеризует «островершинность» распределения в окрестности наивероятнейшего значения.

## 12.4. Пример вычисления

Для случайной величины с рядом распределения:

| $\xi = x_i$ | -4 | -1 | 0 | 5 |
| --- | --- | --- | --- | --- |
| $P(\xi = x_i) = p_i$ | 3/16 | 5/16 | 1/8 | 6/16 |

1. **Начальный момент 2-го порядка:** $\alpha_2(\xi) = \sum x_i^2 p_i = (-4)^2 \cdot \frac{3}{16} + (-1)^2 \cdot \frac{5}{16} + 0^2 \cdot \frac{1}{8} + 5^2 \cdot \frac{6}{16} = \frac{203}{16} = 12,6875$.
2. **Математическое ожидание:** $M(\xi) = \sum x_i p_i = (-4) \cdot \frac{3}{16} + (-1) \cdot \frac{5}{16} + 0 \cdot \frac{1}{8} + 5 \cdot \frac{6}{16} = \frac{13}{16} = 0,8125$.
3. **Центральный момент 3-го порядка:** $\beta_3(\xi) = \sum (x_i - M\xi)^3 p_i \approx 4,72$.
4. **Медиана:** Определяется неоднозначно, любое число $x^* \in (-1, 0)$.
5. **Мода:** $Mo(\xi) = -1$ (так как $p_2 = 5/16$ максимальна среди соседних).
6. **Наивероятнейшее значение:** $Mo^*\xi = 5$ (так как $p_4 = 6/16$ — наибольшая вероятность).
