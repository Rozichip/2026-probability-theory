# Лекция 14. Тестовые непрерывные одномерные случайные величины

## 14.1. Равномерно распределенная случайная величина

**Определение 1.** Случайная величина $\xi$ называется **равномерной** на конечном промежутке $[a, b]$, а соответствующее распределение — равномерным, если её плотность имеет вид:


$$f_{\xi}(x) = \begin{cases} \frac{1}{b-a}, & x \in [a, b], \\ 0, & x \notin [a, b] \end{cases}$$


Здесь $a$ и $b$ — параметры равномерного распределения. График плотности $f_{\xi}(x)$ представляет собой прямоугольник над отрезком $[a, b]$ высотой $\frac{1}{b-a}$.

**Основные числовые характеристики:**

* Математическое ожидание: $M(\xi) = \frac{b+a}{2}$
* Дисперсия: $D(\xi) = \frac{(b-a)^2}{12}$
* Среднее квадратическое отклонение: $\sigma(\xi) = \frac{b-a}{2\sqrt{3}}$

**Интегральная функция распределения $F_{\xi}(x)$:**


$$F_{\xi}(x) = \begin{cases} 0 & \text{при } x \le a \\ \int_{-\infty}^{x} f(t)dt = \int_{a}^{x} \frac{1}{b-a} dt = \frac{x-a}{b-a} & \text{при } a < x \le b \\ 1 & \text{при } x > b \end{cases}$$


График $F_{\xi}(x)$ линейно возрастает от $0$ до $1$ на отрезке $[a, b]$.

**Вероятность попадания в промежуток:**
Для любого подинтервала $(a_1, b_1) \subset [a, b]$ вероятность равна:


$$P(a_1 < \xi < b_1) = \frac{1}{b-a} \int_{a_1}^{b_1} dx = \frac{b_1 - a_1}{b-a}$$


Она зависит только от длины интервала и не зависит от его расположения внутри $[a, b]$.

**Примеры равномерно распределенных величин:**

1. Ошибка, вызванная округлением показаний прибора до целых делений.
2. Расстояние от электростанции до места обрыва кабеля.
3. Время ожидания автобуса при регулярном расписании и случайном приходе пассажира.

**Пример 1.** Поезда метро идут строго по расписанию с интервалом 10 минут. Пассажир приходит в случайный момент времени.
а) Найти среднее время ожидания посадки:
Случайная величина $\xi$ (время ожидания) равномерно распределена на $[0, 10]$.
$M(\xi) = \frac{10+0}{2} = 5$ (мин).
б) Найти вероятность ожидания от 2 до 5 минут:
$P(2 \le \xi \le 5) = P(2 < \xi < 5) = \frac{5-2}{10-0} = 0,3$.

## 14.2. Показательная или экспоненциальная случайная величина

**Определение 2.** Случайная величина $\xi$ распределена по **показательному (экспоненциальному) закону**, если её плотность $f_{\xi}(x)$ имеет вид:


$$f_{\xi}(x) = \begin{cases} \lambda e^{-\lambda x}, & x \ge 0, \\ 0, & x < 0 \end{cases}$$


Здесь $\lambda = const > 0$ — параметр распределения.

**Основные числовые характеристики:**

* Математическое ожидание: $M(\xi) = \frac{1}{\lambda}$
* Дисперсия: $D(\xi) = \frac{1}{\lambda^2}$
* Среднее квадратическое отклонение: $\sigma(\xi) = \frac{1}{\lambda}$

**Интегральная функция распределения $F_{\xi}(x)$:**


$$F_{\xi}(x) = \begin{cases} 0 & \text{при } x \le 0 \\ \int_{0}^{x} \lambda e^{-\lambda t} dt = 1 - e^{-\lambda x} & \text{при } x > 0 \end{cases}$$

**Примеры:**

1. Время между соседними отказами в сложной системе.
2. Промежуток времени между поступлениями вызовов на АТС или в скорую помощь.
3. Время между падениями метеоритов на определенной территории.

**Пример 2.** Время безотказной работы электронной лампы имеет показательное распределение с $\lambda = 0,5$. Найти вероятность, что время работы отклонится от среднего не меньше, чем на $3\sigma(\xi)$.
*Решение.* $M(\xi) = \sigma(\xi) = \frac{1}{0,5} = 2$.
Искомая вероятность: $P(|\xi - M(\xi)| \ge 3\sigma(\xi)) = P(|\xi - 2| \ge 6)$.
Раскрываем модуль: $P(\xi - 2 \ge 6 \cup \xi - 2 \le -6) = P(\xi \ge 8 \cup \xi \le -4) = P(\xi \ge 8) + P(\xi \le -4)$.
Так как $\xi$ не может быть отрицательной, $P(\xi \le -4) = 0$.
$P(\xi \ge 8) = 1 - P(\xi < 8) = 1 - F_{\xi}(8) = 1 - (1 - e^{-4}) = e^{-4} \approx 0,0183$.

## 14.3. Случайная величина, распределенная по нормальному закону (Гаусса)

**Определение 3.** Случайная величина $\xi$ распределена по **нормальному закону**, если её плотность имеет вид:


$$f_{\xi}(x) = \frac{1}{\sigma \sqrt{2\pi}} \exp\left\{-\frac{(x-a)^2}{2\sigma^2}\right\}, \quad -\infty < x < +\infty$$


Параметры: $-\infty < a < +\infty$ и $0 < \sigma < +\infty$. Обозначение: $\xi \sim N(a, \sigma)$.

**Свойства графика $f_{\xi}(x)$:**
График симметричен относительно прямой $x = a$ и имеет холмообразный вид (колокол).

* Параметр $a$ — **«параметр сдвига»** (двигает кривую вправо/влево без изменения формы).
* Параметр $\sigma$ — **«параметр масштаба»** (при увеличении $\sigma$ кривая становится более пологой, при уменьшении — более крутой и высокой).

**Основные числовые характеристики:**

* Математическое ожидание: $M(\xi) = a$
* Дисперсия: $D(\xi) = \sigma^2$
* Среднее квадратическое отклонение: $\sigma(\xi) = \sigma$
* **Срединное отклонение $C(\xi)$:** половина длины интервала с центром в $M(\xi)$, вероятность попадания в который равна $1/2$. Для $N(a, \sigma)$ связь: $C(\xi) \approx 0,675\sigma$.

**Интегральная функция и вероятность попадания в интервал:**


$$F_{\xi}(x) = \frac{1}{2} \left\{ \Phi\left(\frac{x-a}{\sigma}\right) + 1 \right\}$$


где $\Phi(x) = \frac{2}{\sqrt{2\pi}} \int_{0}^{x} e^{-t^2/2} dt$ — **функция Лапласа**.
Свойства $\Phi(x)$: $\Phi(0) = 0$, $\Phi(-x) = -\Phi(x)$, $\Phi(+\infty) = 1$, $\Phi(-\infty) = -1$.
Вероятность попадания в интервал $[c, d)$:


$$P(c \le \xi < d) = \frac{1}{2} \left\{ \Phi\left(\frac{d-a}{\sigma}\right) - \Phi\left(\frac{c-a}{\sigma}\right) \right\}$$

**Правило «трёх сигм»:**
$P(|\xi - a| < 3\sigma) = \Phi(3) \approx 0,9973$.
Вероятность выйти за границы интервала $(a - 3\sigma, a + 3\sigma)$ практически равна нулю ($0,0027$).

**Примеры:** ошибки измерений, координаты попаданий при стрельбе, суммарные характеристики массового производства, сумма большого числа независимых величин (ЦПТ).

**Пример 3.** Ошибка измерения диаметра. Систематическая ошибка $a = 1,2$, стандарт $\sigma = 0,8$. Найти вероятность попадания ошибки в промежуток $[-1,6; 1,6)$.
*Решение.* $\xi \sim N(1,2; 0,8)$.
$P(-1,6 \le \xi < 1,6) = \frac{1}{2} \left[ \Phi\left(\frac{1,6 - 1,2}{0,8}\right) - \Phi\left(\frac{-1,6 - 1,2}{0,8}\right) \right] = \frac{1}{2} [\Phi(0,5) - \Phi(-3,5)] = \frac{1}{2} [\Phi(0,5) + \Phi(3,5)]$.
По таблицам $\Phi(0,5) \approx 0,3829$, $\Phi(3,5) \approx 0,9995$.
$P = \frac{1}{2}[0,3829 + 0,9995] \approx 0,69$.

## 14.4. Распределение хи-квадрат с $r$ степенями свободы (Пирсона)

Пусть $\xi_1, \xi_2, ..., \xi_r$ — независимые случайные величины, каждая из которых имеет стандартное нормальное распределение ($M(\xi_i)=0, \sigma(\xi_i)=1$). Тогда их сумма квадратов распределена по закону **хи-квадрат с $r$ степенями свободы** ($\chi_r^2$).

**Определение 5.** Плотность распределения хи-квадрат имеет вид:


$$f_{\chi_r^2}(x) = \begin{cases} \frac{(1/2)^{r/2}}{\Gamma(r/2)} x^{r/2 - 1} e^{-x/2}, & x > 0 \\ 0, & x \le 0 \end{cases}$$


Где $\Gamma(c) = \int_0^\infty t^{c-1} e^{-t} dt$ — гамма-функция Эйлера ($\Gamma(1)=1, \Gamma(1/2)=\sqrt{\pi}, \Gamma(c+1)=c\Gamma(c)$).

**Основные числовые характеристики:**

* Математическое ожидание: $M(\xi) = r$
* Дисперсия: $D(\xi) = 2r$
* Среднее квадратическое отклонение: $\sigma(\xi) = \sqrt{2r}$

**Применение:** Очень широко применяется в математической статистике для проверки гипотез, например, в критерии согласия «хи-квадрат» Пирсона (для проверки гипотезы о предполагаемом законе неизвестного распределения).
