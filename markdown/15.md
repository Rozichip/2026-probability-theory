# Лекция 15. Многомерные случайные величины (случайные векторы)

## 15.1. Случайные векторы

Ранее мы отдельно рассматривали одномерные случайные величины $\xi(\omega), \eta(\omega), ...$. Следующим шагом является рассмотрение совместных свойств целого набора $\Lambda = (\xi(\omega), \eta(\omega), ...)$ из случайных величин. К таким свойствам относятся функциональные и статистические зависимости между одномерными случайными величинами семейства $\Lambda$.

Необходимо рассматривать набор как упорядоченную систему из нескольких случайных величин, заданных на одном вероятностном пространстве $(\Omega, \mathcal{F}, P(\cdot))$, где:

* $\Omega$ — множество описаний всех элементарных исходов;
* $\mathcal{F}$ — $\sigma$-алгебра наблюдаемых случайных событий эксперимента $E$;
* $P(\cdot)$ — вероятностная функция.

**Пример 1.** Обнаружение летательного аппарата и пуск ракеты. Точка разрыва снаряда определяется тремя случайными координатами $(\xi_1, \xi_2, \xi_3)$.

**Определение 1.** Упорядоченная система из $n$ одномерных случайных величин $\xi_1, \xi_2, ..., \xi_n$, заданных на одном пространстве $(\Omega, \mathcal{F}, P(\cdot))$, называется **случайным вектором** или **многомерной случайной величиной** размерности $n$ и обозначается:


$$\xi(\omega) = (\xi_1(\omega), \xi_2(\omega), ..., \xi_n(\omega))$$


Это отображение: $\xi(\omega): \Omega \to \mathbb{R}^n = \{x=(x_1, ..., x_n) : -\infty < x_i < +\infty\}$.

Поскольку $\{\omega: \xi_1(\omega) < x_1, ..., \xi_n(\omega) < x_n\} = \bigcap_{i=1}^n \{\omega: \xi_i(\omega) < x_i\} \in \mathcal{F}$, можно определить **многомерную интегральную функцию распределения**:


$$F(x_1, x_2, ..., x_n) = P(\{\omega: \xi_1(\omega) < x_1, ..., \xi_n(\omega) < x_n\})$$

Ради наглядности далее будем подробно рассматривать только **двумерные случайные величины** $(n=2, x_1=x, x_2=y)$, так как методы изучения свойств $n$-мерных величин аналогичны.

## 15.2. Двумерная интегральная функция распределения

Двумерная случайная величина $\xi(\omega) = (\xi_1(\omega), \xi_2(\omega)): \Omega \to \mathbb{R}^2$ — это упорядоченная система из двух одномерных случайных величин. Для наглядности их можно отобразить случайной точкой на плоскости.

**Определение.** Двумерная интегральная функция распределения:


$$F(x, y) = P(\{\omega: \xi_1(\omega) < x, \xi_2(\omega) < y\})$$


Это вероятность того, что случайная точка $(\xi_1, \xi_2)$ попадет в область, где $x < x$ и $y < y$.

**Свойства функции $F(x, y)$:**

1. **Теорема 1.** $0 \le F(x, y) \le 1$.
2. **Теорема 2.** $F(x, y)$ — **неубывающая** функция по каждому аргументу. Из $a_2 > a_1 \implies F(a_2, y) \ge F(a_1, y)$.
3. **Формула вероятности в прямоугольнике:**

$$P(\{a_1 \le \xi_1 < a_2, c \le \xi_2 < d\}) = F(a_2, d) - F(a_2, c) - F(a_1, d) + F(a_1, c)$$


4. **Теорема 3.** Предельные соотношения:
* $F(x, -\infty) = F(-\infty, y) = 0$
* $F(+\infty, +\infty) = 1$
* $F(x, +\infty) = F_{\xi_1}(x)$ (маргинальное распределение $\xi_1$)
* $F(+\infty, y) = F_{\xi_2}(y)$ (маргинальное распределение $\xi_2$)



**Важно:** Зная совместную функцию $F(x, y)$, можно найти одномерные (маргинальные) законы $F_{\xi_1}(x)$ и $F_{\xi_2}(y)$, но **обратная задача в общем случае не решается**, так как совместное распределение содержит больше информации.

**Теорема 4.** $F(x, y)$ непрерывна слева по каждому аргументу.

## 15.3. Достаточное условие существования

В многомерном случае теорем 1–4 недостаточно, чтобы функция была интегральной. Необходимым и достаточным условием является выполнение неравенства (из свойства 5):


$$F(b, d) - F(b, c) - F(a, d) + F(a, c) \ge 0 \quad \text{при любых } a < b, c < d$$

**Пример проверки функции $G(u, v)$:**
Пусть $G(u, v) = 0$ при $u+v \le 1$ или $u \le 0$ или $v \le 0$, и $1$ в остальной части.
Эта функция удовлетворяет свойствам 1–4, но проверим свойство 5 для $a=c=1/2$ и $b=d=1$:


$$G(1, 1) - G(1, 1/2) - G(1/2, 1) + G(1/2, 1/2) = 1 - 1 - 1 + 0 = -1 < 0$$


Так как результат $< 0$, функция $G(u, v)$ не может быть интегральной функцией распределения.
