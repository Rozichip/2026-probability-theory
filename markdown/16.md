# Лекция 16. Некоторые типы многомерных случайных величин

## 16.1. Двумерные случайные величины

Теорема Лебега о разложении интегральной функции распределения позволяет провести классификацию одномерных случайных величин, выделяя **дискретные, непрерывные, сингулярные и смешанные** типы.

Многомерные случайные величины более разнообразны по своей структуре, и на текущий момент не удалось провести их полную замкнутую классификацию, аналогичную одномерному случаю.

Ради простоты будем рассматривать случай **двумерной случайной величины** ($n=2$). Результаты легко распространяются на случайные вектора большей размерности.

**Определение 1.** Упорядоченная система из двух одномерных случайных величин $\xi_1(\omega)$ и $\xi_2(\omega)$, заданных на одном вероятностном пространстве $(\Omega, \mathcal{F}, P(\cdot))$, называется **двумерной случайной величиной** (случайным вектором):


$$\xi(\omega) = (\xi_1(\omega), \xi_2(\omega)): \Omega \to \mathbb{R}^2$$

**Интегральная функция распределения** двумерной случайной величины определяется как:


$$F(x, y) = P(\{\omega: \xi_1(\omega) < x, \xi_2(\omega) < y\}), \quad (x, y) \in \mathbb{R}^2$$


Геометрически $F(x, y)$ — это вероятность попадания случайной точки $(\xi_1, \xi_2)$ в заштрихованную область (левее $x$ и ниже $y$).

## 16.2. Двумерная дискретная случайная величина

**Определение 2.** Двумерная случайная величина $\xi(\omega) = (\xi_1(\omega), \xi_2(\omega))$ называется **дискретной**, если образующие её одномерные величины $\xi_1$ и $\xi_2$ являются дискретными.

Пусть $\xi_1$ принимает значения $\{x_1, x_2, ...\}$, а $\xi_2$ принимает значения $\{y_1, y_2, ...\}$.
**Совместное распределение** двумерной дискретной случайной величины задается матрицей вероятностей:


$$r_{i,j} = P(\{\omega: \xi_1(\omega) = x_i, \xi_2(\omega) = y_j\}), \quad \sum_{i}\sum_{j} r_{i,j} = 1$$

**Таблица (матрица) распределения:**

| $\xi_2 \setminus \xi_1$ | $x_1$ | ... | $x_i$ | ... | $x_m$ |
| --- | --- | --- | --- | --- | --- |
| $y_1$ | $r_{1,1}$ | ... | $r_{i,1}$ | ... | $r_{m,1}$ |
| ... | ... | ... | ... | ... | ... |
| $y_j$ | $r_{1,j}$ | ... | $r_{i,j}$ | ... | $r_{m,j}$ |
| ... | ... | ... | ... | ... | ... |
| $y_s$ | $r_{1,s}$ | ... | $r_{i,s}$ | ... | $r_{m,s}$ |

Распределения одномерных величин (маргинальные распределения) находятся суммированием по строкам или столбцам:
$p_i = P(\xi_1 = x_i) = \sum_{j} r_{i,j}$
$q_j = P(\xi_2 = y_j) = \sum_{i} r_{i,j}$

**Важно:** Зная одномерные распределения $\{p_i\}$ и $\{q_j\}$, нельзя восстановить совместное распределение $\{r_{i,j}\}$ (совместное распределение содержит больше информации).

Интегральная функция распределения $F(x, y)$ для дискретной величины:


$$F(x, y) = \sum_{i: x_i < x} \sum_{j: y_j < y} r_{i,j}$$

**Пример 1.** Два игрока независимо бросают по два раза каждый свою несимметричную монету. $P(\text{орел}_1) = 0,7$; $P(\text{орел}_2) = 0,4$.
$\xi_1$ — число выпадений орлов для 1-го игрока ($x \in \{0, 1, 2\}$).
$\xi_2$ — число выпадений орлов для 2-го игрока ($y \in \{0, 1, 2\}$).

После вычисления вероятностей для всех исходов (например, $P(\xi_1=0, \xi_2=0) = P(0,0,0,0) = (1-0,7)^2 \cdot (1-0,4)^2 = 0,0324$), получаем совместное распределение:

**Таблица 1. Совместное распределение**

| $\xi_2 \setminus \xi_1$ | 0 | 1 | 2 |
| --- | --- | --- | --- |
| 0 | 0,0324 | 0,1512 | 0,1764 |
| 1 | 0,0432 | 0,2016 | 0,2352 |
| 2 | 0,0144 | 0,0672 | 0,0784 |

По таблице можно найти маргинальные распределения:

* Сумма по столбцам дает распределение $\xi_1$ ($p_1=0,09; p_2=0,42; p_3=0,49$).
* Сумма по строкам дает распределение $\xi_2$ ($q_1=0,36; q_2=0,48; q_3=0,16$).

Также можно вычислить значения функции $F(x, y)$ для интервалов, например, при $1 < x \le 2$ и $1 < y \le 2$:
$F(x, y) = r_{1,1} + r_{2,1} + r_{1,2} + r_{2,2} = 0,0324 + 0,1512 + 0,0432 + 0,2016 = 0,4284$.
