## Лекция 17. Независимость семейства случайных величин и условные законы распределения

### 17.1. Независимость семейства случайных величин

Вероятностные свойства случайного вектора $(\xi_1, \xi_2)$ исчерпываются свойствами отдельных случайных величин, однако между ними могут существовать различные типы зависимости.

* **Функциональная зависимость**: случайная величина $\xi_2$ однозначно определяется через $\xi_1$ с помощью функции $g(x)$, где $g: \mathbb{R} \to \mathbb{R}$.
* *Пример*: Общая стоимость $\xi_2$ всех вышедших из строя лампочек при заданном их числе $\xi_1$ и постоянной цене.


* **Статистическая (стохастическая) зависимость**: зная значение $\xi_1$, мы можем указать лишь закон распределения $\xi_2$, который зависит от конкретного значения $\xi_1$.
* *Пример*: Вес гриба $\xi_2$ и его высота $\xi_1$ зависимы, но эта связь не является функциональной.


* **Статистическая независимость**: признаки эксперимента на интуитивном уровне считаются причинно несвязанными.

**Определение 1.** Случайные величины $\xi_1, \xi_2, \dots, \xi_n$ называются **статистически независимыми**, если для точки $X=(x_1, x_2, \dots, x_n) \in \mathbb{R}^n$ выполняется равенство:


$$F(x_1, x_2, \dots, x_n) = F_{\xi_1}(x_1) \cdot F_{\xi_2}(x_2) \cdot \dots \cdot F_{\xi_n}(x_n)$$

### 17.2. Условные законы распределения

При $P(B) > 0$ **условная вероятность** вычисляется по формуле:


$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$

**Условная функция распределения** $F_\xi(x|B)$ случайной величины $\xi$ относительно события $B$ вычисляется по формуле:


$$F_\xi(x|B) = \frac{P(\{\xi < x\} \cap B)}{P(B)}$$

* **Для дискретной величины**: при $P(B) > 0$ условное распределение находится по формуле:

$$P(\{\xi = x_j\} | B) = \frac{P(\{\xi = x_j\} \cap B)}{P(B)}$$


* **Для непрерывной величины**: при $P(B) > 0$ **условная плотность распределения** вычисляется как:

$$f_\xi(x|B) = \frac{d(F_\xi(x|B))}{dx}$$



### 17.3. Условные законы при известном значении другой величины

#### Дискретный случай

Пусть $\xi_1, \xi_2$ — дискретные величины с совместным распределением $r_{ij} = P(\{\xi_1 = x_i, \xi_2 = y_j\})$. **Условные законы распределения** $\xi_1$ при условии $\xi_2 = y_j$ определяются соотношением:


$$P(\{\xi_1 = x_i\} | \{\xi_2 = y_j\}) = \frac{r_{ij}}{\sum_{l} r_{lj}}$$

#### Непрерывный случай

Если $\xi_1, \xi_2$ — непрерывные величины с совместной плотностью $f_{\xi_1, \xi_2}(x, y)$, то **условная плотность** $\xi_1$ при условии $\xi_2 = y$ определяется как:


$$f_{\xi_1}(x | \xi_2 = y) = \frac{f_{\xi_1, \xi_2}(x, y)}{\int_{-\infty}^{\infty} f_{\xi_1, \xi_2}(x, y) dx}$$

**Теорема умножения для плотностей**:


$$f_{\xi_1, \xi_2}(x, y) = f_{\xi_2}(y) \cdot f_{\xi_1|\xi_2}(x|y) = f_{\xi_1}(x) \cdot f_{\xi_2|\xi_1}(y|x)$$

### 17.4. Теоремы о полной вероятности и Байеса

Для непрерывной случайной величины $\xi$ с плотностью $f_\xi(x)$:

1. **Формула полной вероятности** для несчётного числа гипотез:

$$P(A) = \int_{-\infty}^{+\infty} P(A|x) f_\xi(x) dx$$


2. **Формула Байеса** для несчётного числа гипотез (если $P(A) > 0$):

$$f_\xi(x|A) = \frac{P(A|x) f_\xi(x)}{\int_{-\infty}^{+\infty} P(A|x) f_\xi(x) dx}$$
