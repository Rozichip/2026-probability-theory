# Лекция 18. Функциональная зависимость между случайными величинами

## 18.1. Понятие о неслучайных функциях от случайных аргументов

На практике часто приходится встречаться с различными случайными величинами, которые связаны функциональной зависимостью с другими случайными величинами.
Например, на одном вероятностном пространстве $(\Omega, \mathcal{F}, P(\cdot))$ могут быть заданы две случайные величины $\xi(\omega)$ и $\eta(\omega)$, для которых справедливо функциональное соотношение вида:


$$\eta(\omega) = g(\xi(\omega))$$


Здесь $g(x)$ — некоторая заданная функция на $\mathbb{R}$ со значениями в $\mathbb{R}$ ($g(x): \mathbb{R} \to \mathbb{R}$). Выражение $\eta = g(\xi)$ называют **неслучайной функцией от случайного аргумента $\xi$**.

В дальнейшем считаем, что функция $g(x)$ задана, и необходимо вычислить неизвестный закон распределения $\eta(\omega)$ по информации о функции $g(x)$ и о законе распределения $\xi(\omega)$.

**Определение 1.** Отображение $g(x): \mathbb{R} \to \mathbb{R}$ называется **измеримым**, если для любого борелевского множества $B_2$ из $\sigma$-алгебры $\mathcal{B}_{\mathbb{R}}$ на действительной прямой справедливо соотношение:


$$\{x : g(x) \in B_2\} = B_1 \in \mathcal{B}_{\mathbb{R}}$$

**Теорема 1.** Пусть задана случайная величина $\xi(\omega): \Omega \to \mathbb{R}$ и измеримая функция $y = g(x): \mathbb{R} \to \mathbb{R}$. Тогда объект $\eta(\omega) = g(\xi(\omega))$ является случайной величиной.
Для произвольной случайной величины $\xi(\omega)$ и измеримой функции $g(x)$ интегральная функция случайной величины $\eta(\omega) = g(\xi(\omega))$ при каждом фиксированном $y \in \mathbb{R}$ находится согласно соотношению:


$$F_{\eta}(y) = P(\{\omega : \xi(\omega) \in g^{-1}(B_y)\}), \quad \text{где } g^{-1}(B_y) = \{x : g(x) < y\}$$

## 18.2. Дискретные одномерные случайные величины и их функциональные зависимости

Если $\xi(\omega)$ — дискретная случайная величина, то $\eta(\omega) = g(\xi(\omega))$ не может принимать больше значений, чем $\xi(\omega)$. Если $\xi(\omega)$ принимает значения $\{x_1, x_2, ...\}$, то возможными значениями $\eta(\omega)$ будут $\{g(x_1), g(x_2), ...\}$. Поскольку некоторые значения могут совпадать, мощность множества значений $\eta$ не больше мощности множества значений $\xi$.

Пусть $q_j = P(\eta = y_j)$ — распределение $\eta$, а $p_i = P(\xi = x_i)$ — распределение $\xi$. Тогда:


$$q_j = P(\eta = y_j) = \sum_{i : g(x_i) = y_j} p_i$$


Интегральная функция $F_{\eta}(y)$ определяется как:


$$F_{\eta}(y) = \sum_{j : y_j < y} q_j = \sum_{j : y_j < y} \sum_{i : g(x_i) = y_j} p_i$$

**Пример 1.** $\xi$ принимает значения $\{-4, -1, 0, 1, 3, 10\}$ с вероятностями $\{1/12, 1/12, 1/6, 1/6, 1/3, 1/6\}$. Найти закон распределения $\eta = \xi^2$.
Значения $\eta$: $y_1=0, y_2=1, y_3=9, y_4=100$.

* $P(\eta=0) = P(\xi=0) = 1/6$
* $P(\eta=1) = P(\xi=-1) + P(\xi=1) = 1/12 + 1/6 = 3/12 = 1/4$
* $P(\eta=9) = P(\xi=-3) + P(\xi=3) = 1/12 + 1/3 = 5/12$ (в файле 1/4, возможно опечатка в исходнике)
* $P(\eta=100) = P(\xi=10) = 1/6$

## 18.3. Непрерывные одномерные случайные величины

Пусть $\xi$ — непрерывная случайная величина с плотностью $f_{\xi}(x)$.
Если функция $y = g(x)$ **строго возрастает** и дифференцируема, то существует обратная функция $x = w(y)$, и плотность распределения $f_{\eta}(y)$ определяется как:


$$f_{\eta}(y) = f_{\xi}(w(y)) \cdot |w'(y)|$$


если $y$ принадлежит области значений $\eta$, и $f_{\eta}(y) = 0$ в противном случае.

Если $g(x)$ — **кусочно-монотонная** и дифференцируемая функция, область определения которой разбита на участки монотонности $D_i$, то на каждом участке $D_i$ находится обратная функция $w_i(y)$ и итоговая плотность:


$$f_{\eta}(y) = \sum_{i=1}^{k} f_{\xi}(w_i(y)) \cdot |w_i'(y)|$$

## 18.4. Многомерные случайные величины и их функциональные зависимости

Для многомерных величин $\xi = (\xi_1, \xi_2)$ и измеримой функции $g(x_1, x_2)$ соотношения аналогичны.

* **Для дискретного случая:**

$$F_{\eta}(z) = P(g(\xi_1, \xi_2) < z) = \sum_{(i,j) : g(x_i, y_j) < z} r_{i,j}$$


* **Для непрерывного случая:**

$$F_{\eta}(z) = \iint_{B^2} f_{\xi_1\xi_2}(x, y) \, dx dy, \quad B^2 = \{(x, y) : g(x, y) < z\}$$



Плотность:

$$f_{\eta}(y) = \int_{-\infty}^{+\infty} f_{\xi_1\xi_2}(x, y-x) \, dx$$



**Свертка:** Если $\xi_1$ и $\xi_2$ независимы, то для суммы $\eta = \xi_1 + \xi_2$ плотность $f_{\eta}(y)$ вычисляется как **свёртка**:


$$f_{\eta}(y) = \int_{-\infty}^{+\infty} f_{\xi_1}(x)f_{\xi_2}(y-x) \, dx$$


Для дискретных независимых величин аналогом является сумма:


$$P(\eta = z_k) = \sum_{i} P(\xi_1 = x_i) P(\xi_2 = z_k - x_i)$$

## 18.5. Композиционная устойчивость

Если суммируемые независимые случайные величины обладают одним и тем же законом распределения (с возможно разными параметрами), и их сумма обладает тем же законом, говорят, что распределение обладает **композиционной устойчивостью**.
Примеры: распределение Пуассона, биномиальное распределение, закон Гаусса.
