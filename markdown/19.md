# Лекция 19. Основные числовые характеристики многомерных случайных величин

## 19.1. Математическое ожидание многомерной случайной величины

Напомним, что $n$-мерной случайной величиной (случайным вектором) называется упорядоченная система из $n$ одномерных случайных величин $\xi = (\xi_1(\omega), \xi_2(\omega), ..., \xi_n(\omega))$, заданных на одном вероятностном пространстве $(\Omega, \mathcal{F}, P(\cdot))$.

Ее вероятностные свойства полностью определяются многомерной интегральной функцией распределения:


$$F(x_1, x_2, ..., x_n) = P(\{\omega : \xi_1(\omega) < x_1, \xi_2(\omega) < x_2, ..., \xi_n(\omega) < x_n\})$$


При больших $n$ или статистической зависимости величин этот объект становится громоздким. Для описания свойств многомерной случайной величины вводят числовые характеристики, такие как математическое ожидание и корреляционные моменты.

**Определение 1.** **Математическим ожиданием** многомерной случайной величины $\xi = (\xi_1, \xi_2, ..., \xi_n)$ называется точка $n$-мерного пространства:


$$M\xi = (M\xi_1, M\xi_2, ..., M\xi_n)$$


где $M\xi_k$ соответствует математическому ожиданию одномерной случайной величины $\xi_k$ для $k=1, ..., n$.

## 19.2. Характеристики разброса многомерной случайной величины

Рассеивание многомерной случайной величины нельзя охарактеризовать вектором из дисперсий $(D_1, D_2, ..., D_n)$, так как разброс должен измеряться неотрицательным числом. Поэтому вводят понятие **ковариации** (корреляционного момента).

**Определение 2.** **Ковариацией** (смешанным центральным моментом второго порядка) величин $\xi_k$ и $\xi_s$ (при $M|\xi_k| < \infty, M|\xi_s| < \infty$) называется:


$$cov(\xi_k, \xi_s) = M[(\xi_k - M\xi_k)(\xi_s - M\xi_s)]$$

**Определение 3.** Две случайные величины называются **некоррелированными**, если $cov(\xi_k, \xi_s) = 0$. Иначе — **коррелированными**.

**Формулы вычисления ковариации:**

* Дискретный случай: $cov(\xi_k, \xi_s) = \sum_{i,j} (x_{k,i} - M\xi_k)(x_{s,j} - M\xi_s) r_{i,j}$
* Непрерывный случай: $cov(\xi_k, \xi_s) = \int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} (x_k - M\xi_k)(x_s - M\xi_s) f(x_k, x_s) \, dx_k dx_s$
* Общий случай: $cov(\xi_k, \xi_s) = \int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} (x_k - M\xi_k)(x_s - M\xi_s) \, dF(x_k, x_s)$

**Ковариационная матрица** $K(\xi)$ для вектора $\xi = (\xi_1, ..., \xi_n)$ состоит из элементов $k_{i,j} = cov(\xi_i, \xi_j)$. Матрица симметрична и неотрицательно определена. Если компоненты попарно некоррелированы, матрица диагональна ($k_{i,i} = D\xi_i$).

**Определение 4.** **Коэффициентом корреляции** называется отношение:


$$corr(\xi_k, \xi_s) = \frac{cov(\xi_k, \xi_s)}{\sigma(\xi_k)\sigma(\xi_s)}$$

**Лемма 1.** $M(\xi_k \cdot \xi_s) = M\xi_k \cdot M\xi_s + cov(\xi_k, \xi_s)$.

**Лемма 2.** $D(\xi_k + \xi_s) = D\xi_k + D\xi_s + 2cov(\xi_k, \xi_s)$.

## 19.3. Дисперсия в направлении единичного вектора

Рассеивание вектора $\xi$ нельзя охарактеризовать одним числом, поэтому рассматривают его **проекцию** на направление единичного вектора $b = (b_1, ..., b_n)$ (где $\sum b_k^2 = 1$).

Проекция равна скалярному произведению: $\varphi(\xi, b) = (\xi, b) = \sum_{k=1}^n b_k \xi_k$.

**Определение 5.** **Дисперсия $D_b\xi$** многомерной случайной величины в направлении единичного вектора $b$ равна дисперсии её проекции:


$$D_b\xi = D\varphi(\xi, b) = \sum_{k=1}^n \sum_{s=1}^n b_k b_s cov(\xi_k, \xi_s)$$

## 19.4. Пример нахождения характеристик

Для двумерной дискретной случайной величины с совместным распределением $r_{i,j}$ (табл. 1):

| $\xi_2 \setminus \xi_1$ | 0 | 1 | 2 |
| --- | --- | --- | --- |
| **0** | 0,0324 | 0,1512 | 0,1764 |
| **1** | 0,0432 | 0,2016 | 0,2352 |
| **2** | 0,0144 | 0,0672 | 0,0784 |

1. Находим частные распределения:
* $\xi_1 \in \{0, 1, 2, 3\}$, вероятности: $p_i = \{125/216, 75/216, 15/216, 1/216\}$.
* $\xi_2 \in \{0, 1, 2, 3\}$, вероятности: $q_j = \{27/216, 81/216, 81/216, 27/216\}$.


2. Математические ожидания: $M\xi_1 = 0,5$, $M\xi_2 = 1,5$. $M\xi = (0,5; 1,5)$.
3. Вычисляем $cov(\xi_1, \xi_2)$:
* $M(\xi_1 \xi_2) = 1$
* $cov(\xi_1, \xi_2) = M(\xi_1 \xi_2) - M\xi_1 M\xi_2 = 1 - 0,5 \cdot 1,5 = 0,25$.


4. Коэффициент корреляции: $corr(\xi_1, \xi_2) = \frac{0,25}{\sigma(\xi_1)\sigma(\xi_2)} \approx 0,447$.
