# Лекция 20. Статистическая зависимость между случайными величинами

## 20.1. Ковариация и статистическая зависимость

На предыдущей лекции для двух случайных величин $\xi_s$ и $\xi_k$ были введены две основные числовые характеристики:

* **Ковариация:** $cov(\xi_s, \xi_k) = M[(\xi_k - M\xi_k)(\xi_s - M\xi_s)]$
* **Коэффициент корреляции:** $corr(\xi_k, \xi_s) = \frac{cov(\xi_k, \xi_s)}{\sigma(\xi_k)\sigma(\xi_s)}$

Обе характеристики описывают смешанный разброс случайных величин, то есть влияние разброса одной величины на разброс другой.

**Лемма 1.** Если $\xi_1$ и $\xi_2$ являются независимыми случайными величинами (и существуют их математические ожидания), то:


$$cov(\xi_1, \xi_2) = 0, \quad corr(\xi_1, \xi_2) = 0$$


*Доказательство:* Для независимых величин $M(\xi_1 \cdot \xi_2) = M\xi_1 \cdot M\xi_2$, следовательно, $cov(\xi_1, \xi_2) = M(\xi_1 \xi_2) - M\xi_1 M\xi_2 = 0$.

**Важно:** 1. $cov(\xi_1, \xi_2) \ne 0$ является достаточным условием зависимости случайных величин.
2. Если $cov(\xi_1, \xi_2) = 0$ и $corr(\xi_1, \xi_2) = 0$ для зависимых величин, это объясняется наличием только **нелинейной зависимости** между ними.

**Лемма 2.** Если $\xi_1$ и $\xi_2$ связаны функциональной линейной зависимостью $\xi_2 = c\xi_1 + b$ ($c \ne 0$), то $cov(\xi_1, \xi_2) = c \cdot D\xi_1 \ne 0$ и $|corr(\xi_1, \xi_2)| = 1$.

* Если $c > 0$, то $corr = 1$ (положительная связь).
* Если $c < 0$, то $corr = -1$ (отрицательная связь).

О характере зависимости можно судить по **диаграмме рассеивания** (точечному графику значений в системе координат $\xi_1, \xi_2$).

## 20.2. Условное математическое ожидание и регрессия

Ковариация теряет смысл как количественная мера зависимости, если величины зависимы, но некоррелированы ($cov = 0$). В таких случаях используют **условные математические ожидания**.

**Определение 1.** Условным математическим ожиданием случайной величины $\xi_1$ относительно $\xi_2 = y$ называется интеграл Стилтьеса:


$$M(\xi_1 | \xi_2 = y) = \int_{-\infty}^{+\infty} x \, dF_{\xi_1}(x | y)$$

**Определение 2.** Множество значений условного математического ожидания называют **регрессией**.

* Регрессия $\xi_2$ по $\xi_1$: $R_{\xi_2|\xi_1}(x) = M(\xi_2 | \xi_1 = x)$
* Регрессия $\xi_1$ по $\xi_2$: $R_{\xi_1|\xi_2}(y) = M(\xi_1 | \xi_2 = y)$

**Пример 1.** Рассматривая урожайность зерна ($\xi_2$) в зависимости от удобрений ($\xi_1$), можно заметить, что средний урожай является функцией от количества удобрений: $M(\xi_2 | \xi_1=1) = 7$ и $M(\xi_2 | \xi_1=2) = 8$.

Для **дискретных** величин регрессия вычисляется как:


$$R_{\xi_2|\xi_1}(x_i) = \frac{\sum_{j} y_j r_{i,j}}{\sum_{j} r_{i,j}}$$


Для **непрерывных** величин:


$$R_{\xi_2|\xi_1}(x) = \int_{-\infty}^{+\infty} y \frac{f(x, y)}{f_{\xi_1}(x)} \, dy$$

График функции регрессии $R_{\xi_2|\xi_1}(x)$ наглядно показывает, как среднее значение одной величины изменяется при росте другой. Если регрессия — прямая линия, связь называется линейной.

## 20.3. Простая линейная регрессия

Простая линейная регрессия оценивает, насколько именно изменится значение одной величины при изменении другой, используя линейное соотношение.

**Уравнение простой линейной регрессии:**


$$y = b_0 + b_1 x \quad (\text{или } \xi_2 = b_0 + b_1 \xi_1)$$

* $b_1$ — коэффициент наклона (тангенс угла наклона прямой).
* $b_0$ — свободный член (пересечение с осью $y$).

Простая линейная регрессия подбирает оптимальную прямую для вычисления зависимой переменной как функции от независимой, что широко используется для **прогнозирования**.

## 20.4. Нелинейная регрессия

Если связь между откликом и независимыми переменными не является линейной, рассматривают **нелинейную регрессию**.
Классы нелинейной регрессии:

1. Нелинейная по независимым переменным (например, полиномиальная регрессия: $y = b_0 + b_1x + b_2x^2 + ...$).
2. Нелинейная по оцениваемому параметру.

## Заключение

1. Ковариация в значительной степени решает вопрос о наличии *линейной* статистической зависимости.
2. Условное математическое ожидание служит мерой зависимости, даже если корреляция равна нулю.
3. Регрессия позволяет описать функциональную зависимость среднего значения одной величины от другой и используется для прогнозирования.
