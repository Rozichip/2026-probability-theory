# Лекция 21. Предельные теоремы теории вероятностей

## 21.1. Массовые случайные явления и их предсказание

Задачей теории вероятностей является построение и изучение математических моделей, порождаемых статистически устойчивыми экспериментами при их многократном проведении. Законы теории вероятностей формулируются на основании адекватного описания реальных статистических закономерностей.

Пусть имеется семейство случайных величин $\{\xi_n; n=1, 2, ...\}$ с неизвестными законами распределения. Первая группа предельных законов описывает ситуации, когда при некоторых общих условиях и большом значении параметра $n$ случайные величины ведут себя как одна хорошо известная или как постоянная величина. Это свойство позволяет осуществлять научные прогнозы и оценивать их точность.

* **Пример 1:** При однократном броске кости мы не можем предсказать результат, но при 1000 бросках чётное число очков выпадет приблизительно в половине случаев ($\approx 500$).
* **Пример 2:** Среднее арифметическое результатов $n$ измерений $n^{-1}(\xi_1 + \dots + \xi_n)$ при большом $n$ приближается к постоянному числу.

Вторая группа предельных законов позволяет аппроксимировать вероятности и законы распределения. Например, в схеме испытаний Бернулли для биномиальной величины при больших $n$ вычисления становятся сложными, что привело к доказательству предельных теорем:

**Локальная предельная теорема Муавра-Лапласа:**
Если $0 < p, q < 1$ и $n \to \infty$, то вероятность $P(B_m)$ можно приближенно вычислить по формуле:


$$P(B_m) \approx \frac{1}{\sqrt{npq}} \cdot \frac{1}{\sqrt{2\pi}} \exp\left\{-\frac{1}{2}\left(\frac{m-np}{\sqrt{npq}}\right)^2\right\}$$

**Интегральная предельная теорема Муавра-Лапласа:**
При $n \to \infty$ вероятность попадания числа успехов в интервал $[a, b)$ аппроксимируется через функцию Лапласа $\Phi(x)$:


$$P(k \le m < s) \approx \frac{1}{2} \left[ \Phi\left(\frac{s-np}{\sqrt{npq}}\right) - \Phi\left(\frac{k-np}{\sqrt{npq}}\right) \right]$$

Третья группа — **центральные предельные теоремы** — позволяет аппроксимировать законы распределения суммы случайных величин. Если $\xi = \xi_1 + \xi_2 + ... + \xi_n$, то при $n \to \infty$ закон распределения суммы стремится к нормальному (даже если законы слагаемых неизвестны). Это важно для оценки суммарных ошибок (измерительных, наблюдателя и т.д.).

## 21.2. Типы сходимости последовательности случайных величин

Физический смысл предельных переходов в теории вероятностей основывается на различных типах сходимости:

**1. Сходимость по вероятности:**
Последовательность $\{\xi_n\}$ сходится по вероятности к $\xi$, если для любого $\epsilon > 0$:


$$\lim_{n \to \infty} P(\{\omega : |\xi_n - \xi| < \epsilon\}) = 1$$


На основе этого типа формулируются **Законы больших чисел**.

**2. Сходимость с вероятностью единица (почти наверное):**


$$\lim_{n \to \infty} P(\{\omega : \lim_{n \to \infty} \xi_n(\omega) = \xi(\omega)\}) = 1$$


На основе этой сходимости формулируются **Усиленные законы больших чисел**.

**3. Сходимость в среднеквадратическом:**


$$\lim_{n \to \infty} M(\xi_n - \xi)^2 = 0$$

**4. Сходимость по распределению (слабая сходимость):**


$$\lim_{n \to \infty} F_{\xi_n}(x) = F_{\xi}(x)$$


в каждой точке непрерывности функции $F_{\xi}(x)$.

* Если $(\xi_n - a_n)/\sigma_n$ сходится по распределению к нормальной величине $(0, 1)$, то последовательность $\xi_n$ называется **асимптотически нормальной**.

## 21.3. Соотношения между типами сходимости

Между типами сходимости существуют следующие иерархические связи:

1. Из сходимости с вероятностью единица следует сходимость по вероятности (обратное неверно).
2. Из сходимости по вероятности следует сходимость по распределению (обратное неверно).
3. Из сходимости в среднеквадратическом следует сходимость по вероятности (обратное неверно).

**Законы больших чисел:**

* **Теорема Чебышева:** Если $\xi_i$ попарно независимы, $M\xi_i = a$ и $D\xi_i = \sigma^2 < \infty$, то $\eta_n = \frac{1}{n} \sum \xi_i$ сходится по вероятности к $a$.
* **Теорема Бернулли:** Относительная частота $n^{-1}\mu(A, n)$ сходится по вероятности к вероятности $P(A)$.

**Центральная предельная теорема (Линдеберга-Леви):**
Если $\xi_i$ независимы и одинаково распределены ($M\xi_i = a, D\xi_i = \sigma^2 > 0$), то при $n \to \infty$ распределение величины $\gamma_n = \frac{\sum \xi_k - na}{\sigma \sqrt{n}}$ стремится к стандартному нормальному распределению:


$$\lim_{n \to \infty} P\left(\frac{\sum \xi_k - na}{\sigma \sqrt{n}} < x\right) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{x} \exp\{-t^2/2\} \, dt$$

*Замечание:* На практике эту теорему применяют при $n \ge 10$.
