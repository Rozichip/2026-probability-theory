# Лекция 21 (доп.). Элементы теории случайных процессов

## 21.1. Определение случайного процесса

До этой лекции рассматривались статистически устойчивые эксперименты $E$ с полностью известными условиями или семейства вероятностно-статистических моделей. При этом роль играли свойства случайных величин или случайных векторов $(\xi_1, \xi_2, ..., \xi_n)$, определяющих скалярный или векторный признак эксперимента $E$.

В эволюционных экспериментах проведение составного эксперимента $E$ представляется в виде последовательности простых экспериментов $E_0, E_1, ..., E_n$. При этом важны как моменты начала проведения каждого эксперимента $t_i$, так и их продолжительность. Фактор времени становится определяющим.

**Определение 1.** **Случайным процессом** называется семейство случайных величин $\{\xi(t) : t \in T\}$, заданных на одном вероятностном пространстве $(\Omega, \mathcal{F}, P(\cdot))$, зависящее от параметра $t \in T$.
Полная запись процесса: $\xi(\omega, t)$.

* Если фиксируем $\omega_0 \in \Omega$, получаем **траекторию (реализацию)** процесса $\xi(\omega_0, t), t \in T$. Элементы пространства состояний $X$ — это значения, которые может принимать процесс.

**Способы задания случайного процесса:**

1. **Поточечный:** задание отображения $\xi(\omega, t): \Omega \times T \to X$.
2. **Конечномерные распределения:** задание системы вероятностей:

$$P(\{\omega : \xi(t_1) \in B_1, \xi(t_2) \in B_2, ..., \xi(t_n) \in B_n\})$$



для любых $t_1, t_2, ..., t_n \in T$ и борелевских множеств $B_1, B_2, ..., B_n$.

## 21.2. Классификация случайных процессов

Классификация проводится по следующим критериям:

1. **Пространство состояний $X$:**
* Дискретное (напр., $\{0, 1, ..., N\}$).
* Непрерывное (напр., $\mathbb{R}$).


2. **Временное пространство $T$:**
* Дискретное (напр., $\{0, 1, ...\}$).
* Непрерывное (напр., $\{t : t \ge 0\}$).


3. **Типы зависимостей:**
* **Процессы с независимыми сечениями:** $\xi(t_1), ..., \xi(t_n)$ независимы в совокупности.
* **Процессы с независимыми приращениями:** приращения $\xi(t_k) - \xi(t_{k-1})$ независимы.
* **Марковские процессы:** процессы с «отсутствием последействия».



**Условие Маркова:**
Процесс называется марковским, если для любых моментов $t_0 < t_1 < ... < t_n < t_{n+1}$ и состояний $a_0, a_1, ..., a_n$:


$$P(\xi(t_{n+1}) \in B_n | \xi(t_0)=a_0, ..., \xi(t_n)=a_n) = P(\xi(t_{n+1}) \in B_n | \xi(t_n)=a_n)$$

## 21.3. Марковские случайные процессы

Если процесс имеет дискретное пространство состояний ($X=\{0, 1, ...\}$) и дискретное время ($T=\{0, 1, ...\}$), он называется **цепью Маркова**.

* **Однородная цепь Маркова:** условные вероятности перехода $p_{i,j}(n) = P(\xi_{n+1} = j | \xi_n = i)$ не зависят от номера шага $n$.
* **Вероятности перехода:** $p_{i,j} = P(\xi_{n+1} = j | \xi_n = i)$.
* **Начальное распределение:** $p_i = P(\xi_0 = i)$.

Все вероятности удовлетворяют условиям: $p_i \ge 0, p_{i,j} \ge 0$ и $\sum p_i = 1, \sum_j p_{i,j} = 1$.

**Уравнение Колмогорова — Чепмена:**


$$p_j^{(n)} = \sum_{i=0}^{\infty} p_i p_{i,j}(n)$$


позволяет находить вероятности состояний на $n$-м шаге.

**Классификация состояний:**

* **Достижимость:** $j$ достижимо из $i$, если есть путь между ними за конечное число шагов.
* **Сообщающиеся состояния ($i \leftrightarrow j$):** если $j$ достижимо из $i$ и $i$ достижимо из $j$.
* **Существенные и несущественные:** состояние $i$ несущественно, если возможен уход из $i$ в $j$ без возможности возврата.
* **Период состояния $d(i)$:** наибольший общий делитель всех $r$, для которых $p_{i,i}(r) > 0$. Если $d(i) > 1$ — периодическое, $d(i) = 1$ — апериодическое.

**Стационарное распределение:**
Цепь называется стационарной, если абсолютные вероятности не меняются со временем:


$$p_j = \sum_{i=0}^{\infty} p_i p_{i,j}$$


Решение этой системы уравнений $x_j = \sum x_i p_{i,j}$ определяет **стационарное распределение**.

**Графическая интерпретация:** Цепь Маркова представляется в виде ориентированного графа, где вершины — состояния, а ребра — направленные дуги с ненулевыми вероятностями перехода.
