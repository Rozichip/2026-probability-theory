# Лекция 23. Статистические (выборочные) законы распределения

## 23.1. Статистическая вероятность

В теории вероятностей мы изучаем теоретические вероятностные характеристики (интегральную функцию распределения $F(x)$, ряд распределения $p_i$, плотность $f(x)$), предполагая известными вероятностную модель $(\Omega, \mathcal{F}, P(\cdot))$.

Математическая статистика ставит перед собой обратную задачу: по результатам многократных наблюдений (выборке) восстановить эти характеристики. Для каждого теоретического объекта существует свой «эмпирический» (статистический) аналог.

**Статистическая вероятность** события $A$ вычисляется по формуле:


$$P_n^*(A) = \frac{\mu(n, A)}{n}$$


где $\mu(n, A)$ — абсолютная частота события $A$ в $n$ независимых испытаниях. Согласно **теореме Бернулли**, $P_n^*(A)$ сходится по вероятности к теоретической вероятности $P(A)$.

Если $\xi$ — дискретная случайная величина, принимающая значения $\{y_1, y_2, ..., y_s\}$, то ее статистическим распределением являются статистические вероятности:


$$P_n^*(B_i) = \frac{\mu(n, B_i)}{n}, \quad B_i = \{\omega : \xi(\omega) = y_i\}$$

## 23.2. Статистическая функция распределения

**Статистическая (эмпирическая) функция распределения** ${F_n}^*(x)$ определяется как статистическая вероятность события $A_x = \{\omega : \xi(\omega) < x\}$:


$${F_n}^*(x) = P_n^*(A_x) = \frac{\mu(n, A_x)}{n}$$


где $\mu(n, A_x)$ — число испытаний, в которых значение случайной величины оказалось строго меньше $x$.

**Свойства:**

* ${F_n}^*(x)$ обладает всеми четырьмя основными свойствами теоретической функции распределения $F(x)$.
* Согласно теоремам Гливенко-Кантелли и Колмогорова, при увеличении объема выборки $n \to \infty$, случайная функция ${F_n}^*(x)$ сходится к теоретической $F(x)$. Оценка $F(x) \approx {F_n}^*(x)$ является равномерной с точностью до величины порядка $1/\sqrt{n}$.

**Построение ${F_n}^*(x)$:**

1. **По вариационному ряду:** $x_1^* \le x_2^* \le ... \le x_n^*$:

$${F_n}^*(x) = \frac{k}{n} \quad \text{при } x \in (x_k^*, x_{k+1}^*]$$


2. **По статистическому ряду:** В точках разрыва $y_j$ функция имеет скачок величиной $n_j/n$, где $n_j$ — частота значения $y_j$.

## 23.3. Гистограмма и полигон частот

### Гистограмма (${f_n}^*(x)$)

Строится по **группированному статистическому ряду** (информационной совокупности):

1. Весь диапазон выборки разбивается на $k$ разрядов $\Pi_j = [a_{j-1}, a_j)$.
2. Для каждого разряда вычисляется абсолютная частота $\nu_j$ и относительная $\nu_j/n$.
3. Плотность (высота прямоугольника): $h_j = \frac{\nu_j}{n(a_j - a_{j-1})}$.

Аналитическая запись:


$$f_n^*(x) = \frac{\nu_j}{n(a_j - a_{j-1})} \quad \text{при } a_{j-1} \le x < a_j$$

### Полигон частот

Визуализация статистического ряда или группированного ряда в виде ломаной линии:

* Для дискретной величины: соединяются точки $(y_j, \nu_j)$ или $(y_j, \nu_j/n)$.
* Для непрерывной величины: соединяются точки $(b_j, f_n^*(b_j))$, где $b_j$ — середины разрядов.

## 23.4. Пример нахождения характеристик

Пусть задана выборка (масса клетчатки, потребляемая 30 респондентами).

1. **Вариационный ряд:** упорядочивание данных от 60 до 390. Размах выборки $R = 390 - 60 = 330$.
2. **Группировка:** при $n=30$ количество разрядов $k \approx 6$, ширина $\Delta = 66$.
3. **Группированный ряд:**

| Разряд $\Pi_j$ | Частота $\nu_j$ | Относ. частота $p_j^*$ |
| --- | --- | --- |
| $[27, 93)$ | 2 | 2/30 |
| $[93, 159)$ | 5 | 5/30 |
| $[159, 225)$ | 12 | 12/30 |
| $[225, 291)$ | 8 | 8/30 |
| $[291, 357)$ | 2 | 2/30 |
| $[357, 423)$ | 1 | 1/30 |

По полученной статистической плотности можно оценивать вероятности: $P(159 \le \xi < 225) \approx (225-159) \cdot 0,006 = 0,396$.
