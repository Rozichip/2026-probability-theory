# Лекция 24. Статистические (выборочные) числовые характеристики

## 1. Статистические числовые характеристики одномерной случайной величины

Теоретические вероятностные характеристики (математическое ожидание, дисперсия, моменты и т.д.) имеют свои аналоги в математической статистике, которые называются **выборочными (статистическими, эмпирическими)**. Они строятся на основе реализации повторной выборки $(x_1, x_2, \dots, x_n)$, полученной для случайной величины $\xi$.

### 1.1. Выборочное математическое ожидание

**Выборочное математическое ожидание** (выборочное среднее) есть среднее арифметическое значений выборки:


$$M^*(\xi) = \frac{1}{n} (x_1 + x_2 + \dots + x_n) = \frac{1}{n} \sum_{i=1}^n x_i$$

### 1.2. Выборочные дисперсия и стандарт

* **Выборочная дисперсия:**

$$D^*(\xi) = \frac{1}{n} \sum_{i=1}^n (x_i - M^*\xi)^2$$


* **Несмещенная выборочная дисперсия:**

$$\tilde{D}(\xi) = \frac{1}{n-1} \sum_{i=1}^n (x_i - M^*\xi)^2$$


* **Выборочное среднее квадратическое отклонение:**

$$\sigma^*(\xi) = \sqrt{D^*(\xi)}$$



### 1.3. Выборочные моменты

* **Начальный момент $k$-го порядка ($k=0, 1, \dots$):**

$$\alpha_k^*(\xi) = \frac{1}{n} \sum_{i=1}^n (x_i)^k$$


* **Центральный момент $k$-го порядка ($k=0, 1, \dots$):**

$$\beta_k^*(\xi) = \frac{1}{n} \sum_{i=1}^n (x_i - M^*\xi)^k$$



### 1.4. Выборочные мода, медиана и квантили

* **Выборочная медиана ($Me^*\xi$):** Определяется по вариационному ряду $x_1^* \le x_2^* \le \dots \le x_n^*$:
* Если $n = 2k+1$: $Me^*\xi = x^*_{k+1}$.
* Если $n = 2k$: $Me^*\xi = \frac{x^*_k + x^*_{k+1}}{2}$.


* **Выборочная мода ($Mo^*\xi$):** Значение, которое встречается чаще других в статистическом ряду.
* **Выборочный квантиль порядка $p$ ($K_p^*\xi$):** Число, удовлетворяющее неравенству:

$$F_n^*(K_p^*\xi) \le p \le F_n^*(K_p^*\xi + 0)$$



## 2. Многомерные статистические числовые характеристики

Для двумерного случайного вектора $\xi = (\xi_1, \xi_2)$ выборка имеет вид пар $(x_1, y_1), \dots, (x_n, y_n)$.

### 2.1. Статистическое математическое ожидание

$$M^*\xi = (M^*\xi_1, M^*\xi_2)$$


Вычисляется как выборочные средние каждой компоненты вектора отдельно.

### 2.2. Выборочные ковариация и коэффициент корреляции

* **Выборочная ковариация:**

$$cov^*(\xi_1, \xi_2) = \frac{1}{n} \sum_{i=1}^n (x_i - M^*\xi_1)(y_i - M^*\xi_2)$$



(Также существует несмещенная оценка с делением на $n-1$).
* **Выборочный коэффициент корреляции:**

$$corr^*(\xi_k, \xi_s) = \frac{cov^*(\xi_k, \xi_s)}{\sigma^*(\xi_k) \sigma^*(\xi_s)}$$


* **Выборочная ковариационная матрица $K^*(\xi)$:**

$$K^*(\xi) = \begin{pmatrix} cov^*(\xi_1, \xi_1) & cov^*(\xi_1, \xi_2) \\ cov^*(\xi_2, \xi_1) & cov^*(\xi_2, \xi_2) \end{pmatrix} = \begin{pmatrix} D^*\xi_1 & cov^*(\xi_1, \xi_2) \\ cov^*(\xi_2, \xi_1) & D^*\xi_2 \end{pmatrix}$$



### 2.3. Дисперсия в направлении единичного вектора

Для вектора $\xi = (\xi_1, \dots, \xi_n)$ и единичного вектора $b = (b_1, \dots, b_n)$ дисперсия вычисляется как:


$$D^*_b\xi = \sum_{k=1}^n \sum_{s=1}^n b_k b_s cov^*(\xi_k, \xi_s)$$

## Пример нахождения (социально-экономические показатели)

Для 6 наблюдений по двум показателям ($\xi_1$ — численность пенсионеров, $\xi_2$ — доходы):

1. **Математическое ожидание:** $M^*\xi = (1038,1; 27553,4)$.
2. **Ковариация:** $cov^*(\xi_1, \xi_2) = 95271,7$.
3. **Дисперсии:** $D^*\xi_1 = 304,0$; $D^*\xi_2 = 31550081,4$.
4. **Корреляция:** $corr^*(\xi_1, \xi_2) \approx 0,97$ (сильная линейная связь).

**Графическая интерпретация:**

* Диаграмма рассеяния показывает близость к прямой линии, что подтверждает высокий коэффициент корреляции.
* Медиана в примере равна 210 (при $n=30$ это полусумма 15-го и 16-го членов ряда).
* Мода определяется неоднозначно (множество значений).

---

### Заключение

1. Все теоретические вероятностные характеристики (ожидание, дисперсия, моменты) имеют статистические аналоги, вычисляемые по выборке.
2. Статистические характеристики обладают свойствами, аналогичными теоретическим.
3. Ковариация и корреляция позволяют оценивать степень линейной зависимости многомерных величин.
4. Понятие дисперсии в направлении единичного вектора позволяет характеризовать разброс многомерной величины как единого целого.
