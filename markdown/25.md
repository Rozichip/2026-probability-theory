# Лекция 25. Точечное оценивание неизвестных параметров распределения. Критерии качества статистических оценок

## 1.1. Понятие оценки неизвестного параметра

Пусть случайная величина $\xi$ имеет распределение $P_{\xi, \theta}(\cdot) = P_\theta(\cdot)$, определяемое некоторым неизвестным параметром $\theta \in \Theta \subset \mathbb{R}$. Распределение случайной величины принадлежит параметрическому семейству $\rho = \{P_\theta(\cdot) : \theta \in \Theta\}$ вероятностных функций.

Вероятностная функция $P_\theta(\cdot)$ может быть задана:

* Интегральной функцией распределения $F(x; \theta)$, определенной при $-\infty < x < +\infty$.
* Плотностью распределения вероятностей $f(x; \theta)$ для непрерывной случайной величины.
* Распределением вероятностей $\{P(\xi=y_j; \theta) = p(y_j; \theta), j=1, 2, ...\}$ для дискретной случайной величины (набором вероятностей для всех возможных значений).

**Задача оценивания:**
Предположим, требуется оценить неизвестный параметр $\theta$ по результатам $n$ независимых опытов. Обозначим результат $i$-го опыта через $\xi_i$. Мы получаем $n$ копий случайной величины $\xi$ или **повторную выборку** $(\xi_1, \xi_2, ..., \xi_n)$ объема $n$. После проведения опытов получаем значение повторной выборки $(x_1, x_2, ..., x_n)$.

Задача сводится к определению конкретной вероятностной функции $P_\theta(\cdot)$ из параметрического семейства по наблюдаемым выборочным значениям $(x_1, x_2, ..., x_n)$. Поскольку $P_\theta(\cdot)$ однозначно определяется значением параметра $\theta$ (скалярного или векторного), возникает задача получения оценки для значения неизвестного параметра.

* *Пример:* Если $\xi \sim \text{Пуассон}(\lambda)$, то $\theta = \lambda = M\xi$. Если $\xi \sim \text{Нормальное}(a, \sigma^2)$, то $\theta = (a, \sigma^2)$, где $a = M\xi$, $\sigma^2 = D\xi$.

**Определение 1.** Любую измеримую функцию $\theta_n^* = \theta_n^*(\xi_1, \xi_2, ..., \xi_n)$ от $n$ случайных величин будем называть **статистикой**.
**Определение 2.** **Оценкой** неизвестного параметра $\theta$ назовем любую статистику $\theta_n^*$, множество значений которой совпадает с множеством значений оцениваемого параметра $\theta$.

Так как оценка является измеримой функцией случайных величин, она сама является случайной величиной. Её закон распределения зависит от закона распределения $\xi$ (и параметра $\theta$) и от числа опытов $n$. Поскольку для одного параметра можно предложить множество оценок, вводятся требования к их «доброкачественности».

## 2.1. Основные требования к оценкам

Для того чтобы оценка была качественной, необходимо выполнение требований **несмещенности, состоятельности и эффективности**.

**Определение 3 (Несмещенность).** Если для всех $n=1, 2, ...$ выполняется условие $M_\theta(\theta_n^*) = \theta$ при каждом $\theta \in \Theta$, то оценка $\theta_n^*$ называется **несмещенной**.

**Лемма 1.** Статистическое математическое ожидание $M^*\xi$ является несмещенной оценкой математического ожидания $M\xi = \theta$ случайной величины $\xi$.
*Доказательство:* $M(M^*\xi) = M(n^{-1} \sum \xi_i) = n^{-1} \sum M\xi_i = n^{-1} (n \cdot \theta) = \theta$.

**Лемма 2.** Статистическая дисперсия $D^*\xi = \frac{1}{n} \sum (\xi_i - M^*\xi)^2$ является **смещенной** оценкой теоретической дисперсии $D\xi = \theta_2$.
*Доказательство:* Вычисление дает $M(D^*\xi) = \frac{n-1}{n} \theta_2 \ne \theta_2$.
Оценка является лишь *асимптотически несмещенной*, так как $\lim_{n \to \infty} M(D^*\xi) = \theta_2$.
Для получения несмещенной оценки используют **несмещенную выборочную дисперсию**:


$$\tilde{D}(\xi) = \frac{1}{n-1} \sum_{i=1}^n (\xi_i - M^*\xi)^2$$

*Пример 1:* Для распределения Пуассона $P(\xi=x) = \frac{\theta^x}{x!} e^{-\theta}$ несмещенных оценок для параметра $\theta$ не существует (при одном наблюдении).

**Определение 4 (Состоятельность).** Оценка $\theta_n^*$ называется **состоятельной**, если при $n \to \infty$ она сходится по вероятности к оцениваемому параметру $\theta$.

**Теорема 1.** Если оценка $\theta_n^*$ является асимптотически несмещенной и её дисперсия стремится к нулю при $n \to \infty$, то оценка $\theta_n^*$ является состоятельной.

**Лемма 3.** Статистическое математическое ожидание $M^*\xi$ является состоятельной оценкой математического ожидания $M\xi$.

**Эффективность:**
Если существует несколько несмещенных и состоятельных оценок, выбирают ту, у которой **наименьшая дисперсия**.

**Определение 5.** Несмещенная оценка $\theta_{3,n}^*$ называется **эффективной**, если она имеет наименьшую дисперсию среди всех возможных несмещенных оценок того же объема $n$. Эффективная оценка существует не всегда.

**Эффективность произвольной оценки** $\theta_n^*$:


$$e(\theta_n^*) = \frac{D_\theta(\theta_{3,n}^*)}{D_\theta(\theta_n^*)}$$


Оценка называется **асимптотически эффективной**, если $\lim_{n \to \infty} \frac{D_\theta(\theta_{3,n}^*)}{D_\theta(\theta_n^*)} = 1$.

**Среднеквадратическая ошибка:** $M_\theta(\theta_n^* - \tau(\theta))^2$. Критерий сравнения: если ошибка одной оценки меньше, чем другой, то эта оценка «лучше».
