# Лекция 26. Методы точечного оценивания неизвестных параметров законов распределения случайных величин

## 1. Метод максимального правдоподобия (ММП)

Метод основан на максимизации функции правдоподобия.

**Определение:** **Функция правдоподобия** $L(x; \theta)$ — это совместное распределение выборочных данных $x = (x_1, x_2, ..., x_n)$, рассматриваемое как функция параметра $\theta$.

* В непрерывном случае используется совместная плотность: $L(x; \theta) = f(x_1; \theta) \cdot f(x_2; \theta) \cdot ... \cdot f(x_n; \theta)$.
* В дискретном случае используется совместная вероятность.

**Оценка максимального правдоподобия (ОМП)** $\theta^*(x)$ — это такое значение $\theta \in \Theta$, при котором $L(x; \theta)$ достигает максимума:


$$L(x; \theta^*(x)) = \sup_{\theta \in \Theta} \{L(x; \theta)\}$$

Если функция дифференцируема и максимум достигается во внутренней точке, ОМП находится из **уравнения правдоподобия**:


$$\frac{\partial L(x; \theta)}{\partial \theta} = 0 \quad \text{или} \quad \frac{\partial \ln L(x; \theta)}{\partial \theta} = 0$$


Для векторного параметра $\theta = (\theta_1, ..., \theta_k)$ решается система:


$$\frac{\partial \ln L(x; \theta)}{\partial \theta_j} = 0, \quad j=1, ..., k$$

*Свойство:* Оценки максимального правдоподобия являются состоятельными и асимптотически эффективными.

### Пример 1 (Нормальное распределение $N(a, \sigma^2)$)

Дана выборка $n=40$ (рост школьников). Параметры $\theta_1 = a = M\xi$, $\theta_2^2 = \sigma^2 = D\xi$.
Функция правдоподобия:


$$L(x; \theta_1, \theta_2^2) = \frac{1}{(2\pi)^{n/2} (\theta_2)^n} \exp\left\{ -\sum_{i=1}^n \frac{(x_i - \theta_1)^2}{2\theta_2^2} \right\}$$


После логарифмирования и решения системы уравнений правдоподобия получены ОМП:

* $\theta_1^* = \frac{1}{n} \sum x_i = M^*\xi$ (выборочное среднее)
* $(\theta_2^2)^* = \frac{1}{n} \sum (x_i - \theta_1^*)^2 = D^*\xi$ (статистическая дисперсия)

Для данной выборки ($n=40$): $\theta_1^* = 165,2$; $(\theta_2^2)^* = 334,2$.

## 2. Метод моментов

**Принцип:** Оценки неизвестных параметров находятся путем приравнивания теоретических моментов (начальных или центральных) к соответствующим выборочным моментам.

Количество уравнений должно быть равно количеству оцениваемых параметров $k$. Система уравнений:


$$\begin{cases} \alpha_j(\theta_1, ..., \theta_k) = \alpha_j^* \\ j = 1, ..., k \end{cases}$$


где $\alpha_j^*$ — выборочный начальный момент $j$-го порядка ($\alpha_j^* = \frac{1}{n} \sum x_i^j$), а $\alpha_j(\dots)$ — теоретический момент.

* *Свойства:* Оценки метода моментов всегда являются состоятельными. Однако они часто уступают ММП по асимптотической эффективности (асимптотическая эффективность метода моментов всегда $\le 1$, и часто $< 1$).

### Пример 2 (Равномерное распределение на $[a, b]$)

Выборка $n=200$. Параметры: $a, b$.
Система уравнений:

1. $M\xi = \frac{a+b}{2} = M^*\xi$
2. $D\xi = \frac{(b-a)^2}{12} = D^*\xi$

Для выборки вычислены: $M^*\xi = 11,3$; $D^*\xi = 32,9$.
Система:


$$\begin{cases} \frac{a+b}{2} = 11,3 \\ \frac{(b-a)^2}{12} = 32,9 \end{cases}$$


Решение: $a^* \approx 1,4$; $b^* \approx 21,3$.

### Заключение

1. Рассмотрены два основных метода точечного оценивания: метод максимального правдоподобия и метод моментов.
2. Метод максимального правдоподобия позволяет получать «наилучшие» оценки (состоятельные и асимптотически эффективные).
3. Метод моментов проще в вычислении и дает состоятельные оценки, но они менее эффективны, чем оценки ММП.
4. Приведены примеры применения методов для нормального и равномерного распределений.
