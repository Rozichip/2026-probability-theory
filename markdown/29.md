# Лекция 29. Проверка сложных статистических гипотез о виде распределения

## 1. Проверка сложных статистических гипотез критерием хи-квадрат Пирсона-Фишера

**Основные понятия:**

* **Простая гипотеза $H_0$:** однозначно фиксирует распределение вероятностей.
* **Сложная гипотеза $H_0$:** не фиксирует распределение однозначно (гипотетическая функция распределения содержит неизвестные параметры).
* На практике случай полностью определенного распределения встречается редко, поэтому задача проверки сложных гипотез является основной.

Пусть по выборке $x = (x_1, \dots, x_n)$ нужно проверить гипотезу $H_0$: функция распределения $F_\xi(x)$ совпадает с $F_0(x; \theta)$, где параметр $\theta = (\theta_1, \dots, \theta_k)$ неизвестен.
Множество всех допустимых интегральных функций в рамках гипотезы $H_0$:


$$\rho_\xi(H_0) = \{F_0(x; \theta) : \theta \in \Theta \subseteq \mathbb{R}^k\}$$

### Алгоритм проверки:

1. **Разбиение выборки:** Область значений случайной величины разбивается на $r$ непересекающихся групп $B_1, \dots, B_r$.
2. **Расчет вероятностей:** Вычисляются теоретические вероятности $p_j(\theta) = P(\xi \in B_j | F_0(x; \theta))$.
3. **Расчет частот:** Вычисляются абсолютные частоты $\nu_j$ попадания элементов выборки в каждый интервал $B_j$ ($\sum_{j=1}^r \nu_j = n$).

Если бы параметр $\theta$ был известен, мы использовали бы стандартную статистику:


$$t_n^*(\xi; \theta) = \sum_{j=1}^r \frac{n}{p_j(\theta)} \left( \frac{\nu_j}{n} - p_j(\theta) \right)^2$$


Но так как $\theta$ неизвестен, его необходимо оценить по выборке.

**Критерий Пирсона-Фишера:**
Роберт Фишер предложил использовать оценки параметров (например, методом максимального правдоподобия) и в предельном распределении статистики $t_n^*$ уменьшить число степеней свободы на количество оцениваемых параметров $k$.

* Предельное распределение статистики $t_n^*$ — это **хи-квадрат ($\chi^2$)** с $r - k - 1$ степенями свободы.

**Методы оценки параметров:**

* Метод максимального правдоподобия.
* Метод минимума хи-квадрат (система уравнений):

$$\sum_{j=1}^r \frac{\nu_j}{p_j} \frac{\partial p_j}{\partial \theta_i} = 0, \quad i = 1, \dots, k$$

## 2. Пример проверки сложной гипотезы (Распределение Пуассона)

**Задача:** Проверить, подчиняется ли количество красных кровяных шариков в отделении гемацитометра распределению Пуассона ($n=169$ наблюдений).
Гипотеза $H_0$: $\xi \sim \text{Пуассон}(\theta)$ с неизвестным $\theta$.

1. **Группировка:** Имеющиеся данные (от 0 до 21 и более) группируются в $r=11$ разрядов (последние интервалы объединяются, чтобы частоты были не менее 9-10).
2. **Оценка параметра:** Используется уравнение для метода моментов или ММП для параметра $\theta$ (в примере получается специфическая система уравнений для параметра Пуассона).
3. **Расчет статистики:** Вычисляются ожидаемые частоты $n p_j(\theta^*)$ и сравниваются с наблюдаемыми $\nu_j$.
Статистика $t_n^* = 4,006$.
4. **Критерий:**
* Число степеней свободы: $r - k - 1 = 11 - 1 - 1 = 9$.
* Уровень значимости $\alpha = 0,1$.
* По таблицам $\chi^2$ при 9 степенях свободы находим порог $t_\alpha = 14,684$.


5. **Вывод:** Так как $4,006 < 14,684$, гипотеза $H_0$ о пуассоновском распределении принимается (статистически значимых расхождений нет).
