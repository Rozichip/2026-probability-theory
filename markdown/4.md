# Лекция 4. Классический, геометрический и эмпирический подходы к вычислению вероятностей случайных событий

### 4.1. Вычисление вероятности для опытов с конечным множеством равновозможных элементарных исходов

Субъективный подход даёт различные числа $P_{c}(A)$ (лекция 3) при его использовании исследователями для измерения шанса наступления случайного события $A$.
Поэтому естественно желание найти подходы, которые измеряли бы шанс наступления случайного события $A$ в виде одного и того же числа $P(A)$ независимо от субъекта.
Как правило, для этого рассматривают некоторые группы экспериментов, которые выделяются с помощью тех или иных аксиом или ограничений.

Для совокупности **классических экспериментов** имеет место два ограничения:

1. конечное число элементарных исходов, каждый из которых можно наблюдать;
2. симметрия эксперимента $E$, которая позволяет на интуитивном уровне утверждать о равновозможных шансах наступления каждого из элементарных исходов.

Пусть некоторому классическому эксперименту поставлена в соответствие теоретико-множественная модель $(\Omega, \mathcal{F})$.
Тогда, согласно ограничениям 1) и 2), будем иметь конечное множество **равновозможных элементарных исходов** $\mathfrak{A}^{\prime} = \{A^{\prime}_{1}, A^{\prime}_{2}, ..., A^{\prime}_{n}\}$ и соответствующее множество описаний элементарных исходов $\Omega = \{\omega_{1}, \omega_{2}, ..., \omega_{n}\}$.
Причем элементарные события $A^{\prime}_{k} = \{\omega_{k}\}$, $k=1, 2, ..., n$ образуют полную группу несовместимых равновозможных событий.
При $m \le n$ любое случайное событие $A$ представимо в виде:


$$A = \{\omega_{a_{1}}, \omega_{a_{2}}, ..., \omega_{a_{m}}\} = \{\omega_{a_{1}}\} \cup \{\omega_{a_{2}}\} \cup ... \cup \{\omega_{a_{m}}\}$$

**Определение 1.** Если $A = \{\omega_{a_{1}}, \omega_{a_{2}}, ..., \omega_{a_{m}}\}$, $m \le n$ и все элементарные случайные события множества $\Omega = \{\omega_{1}, \omega_{2}, ..., \omega_{n}\}$ равновозможны, то **вероятность** наступления события $A$ находится по формуле (классическая формула вероятности):


$$P(A) = \frac{m}{n} = \frac{|A|}{|\Omega|}$$


Здесь:

* $m$ — число элементарных случайных событий, благоприятствующих наступлению события $A$ (мощность $A$);
* $n$ — общее число элементов в множестве $\Omega$ (мощность $\Omega$).

**Пример 1 (задача Даламбера).** Симметричная монета непреднамеренно бросается два раза на некоторую поверхность стола. Требуется найти вероятность наступления события $A$ — {хотя бы раз появится герб}.

*Решение Даламбера:* Герб появится либо при первом бросании и в этом случае второе бросание не нужно, либо только при втором броске, либо герб совсем не выпадает.
Всего элементарных исходов три, то есть $|\Omega| = 3$. Из них благоприятствуют ожидаемому событию $A$ только первые два, то есть $|A| = 2$. Следовательно, искомая вероятность события $A$ будет равна $P(A) = |A|/|\Omega| = 2/3$.
Но практика показывала, что относительная частота $\mu(A, N)/N$ для события $A$ при неограниченном увеличении $N$ группируется как угодно близко около числа $3/4 = 0,75$.
Например, при 1000 повторов опыта, в 752 случаях появлялось событие $A$. То есть $\mu(A, N)/N = 752/1000 = 0,752 \approx 0,75$.
Возникло противоречие теоретического решения с результатом опыта.

*Правильное решение задачи Даламбера:*
Строим точную математическую модель эксперимента. Определим комплекс условий $\Sigma = \{u_{1}, u_{2}, u_{3}, u_{4}, u_{5}\}$, где $u_{1}$ — выбрана симметричная монета определённого достоинства, $u_{2}$ — выбран механизм бросания монеты, $u_{3}$ — монета бросается два раза, $u_{4}$ — выбрана поверхность стола, $u_{5}$ — выбрана достаточная освещенность для определения выпавшей стороны монеты и т. д.
Выберем элементарные исходы и их описания:

* $\omega_{1} = (1, 1)$ — выпадает герб при первом и втором бросках;
* $\omega_{2} = (1, 0)$ — выпадает герб при первом броске и решётка при втором;
* $\omega_{3} = (0, 1)$ — выпадает решётка при первом броске и герб при втором;
* $\omega_{4} = (0, 0)$ — выпадает решётка при первом и втором бросках.
Тогда множество $\Omega = \{\omega_{1}, \omega_{2}, \omega_{3}, \omega_{4}\}$ и $|\Omega| = 4$.

Выразим случайное событие $A$ {хотя бы раз появится герб} через элементы множества $\Omega$.
Получаем $A = \{\omega_{1}, \omega_{2}, \omega_{3}\}$, и, следовательно, $|A| = 3$.
Согласно определению классической вероятности $P(A) = |A|/|\Omega| = 3/4$. Что и было подтверждено при практических исследованиях.

*Ошибки Даламбера:*

1. Нарушено в действительности не только условие проведения эксперимента, но и **условие симметрии** его проведения (если герб появился при первом броске, отказывались от второго).
2. Выбранные по Даламберу исходы **не равновозможны**, т.е. имеют разные шансы наступления, и применять классическое определение нельзя.

### 4.2. Вычисление вероятности для опытов с несчётным числом равновозможных исходов

В жизни встречаются эксперименты, которым соответствует **несчётное (бесконечное)** число элементарных равновозможных исходов и такое же число элементов в множестве $\Omega$.
В таком случае нельзя применять классическую формулу для вычисления вероятности $P(A) = m/n = |A|/|\Omega|$, так как в знаменателе дроби будет стоять бесконечность ($|\Omega| = \infty$).
Чтобы выйти из такой непростой ситуации, придумали альтернативный способ вычисления — **геометрический**.

Пусть реальный эксперимент $E$ таков, что множество $\Omega$ описаний всех элементарных равновозможных исходов можно представить некоторой **областью $G$** в одном из видов:

* Область $G$ — некоторый промежуток на действительной оси, имеющий положительную длину ($mes$), например отрезок $G = [a, b]$;
* Область $G$ — некоторая геометрическая фигура на плоскости, имеющая положительную площадь ($mes$), например прямоугольник $G = \{(x, y): a \le x \le b, c \le y \le d\}$;
* Область $G$ — некоторая трёхмерная фигура в пространстве, имеющая положительный объём ($mes$), например параллелепипед $G = \{(x, y, z): a \le x \le b, c \le y \le d, u \le z \le v\}$.
Где $a, b, c, d, u, v$ — фиксированные действительные числа.

Вместо реального эксперимента $E$ рассматривается адекватный ему **модельный эксперимент $E_{m}$**, который заключается в том, что в область $G$ **непреднамеренно** бросается точка.
Термин «непреднамеренно» означает, что ни одному положению точки внутри области $G$ не отдаётся предпочтение. Этим обеспечивается симметрия экспериментов $E$ и $E_{m}$.
Взаимно однозначное соответствие: случайный выбор исхода с описанием $\omega \in \Omega$ соответствует выбору точки $x \in G$. Исход $A$ соответствует некой области $g \subset G$.
Теоретико-множественная модель: $\Omega = G$, $\mathcal{F} = \{g: g \subset G, mes(g) \text{ существует}\}$.

**Определение 2.** Вероятность наступления события $A$ реального эксперимента с несчётным числом равновозможных элементарных случайных событий находится по **геометрической формуле**:


$$P(A) = \frac{mes(g)}{mes(G)}$$


Здесь $mes$ — мера области (длина, площадь, объем и т.д.).

**Пример 2 (Задача о встрече).** Два лица А и В условились встретиться в определенном месте между 12 часами и часом дня. Пришедший первым ждет другого в течении 20 минут, после чего уходит. Требуется найти вероятность встречи лиц А и В, если приход каждого из них в течение указанного часа равновозможен и моменты прихода независимы.
*Решение:* Случайными являются моменты прихода лиц А и В (в часах) — $x$ и $y$.
Множество $\Omega = \{\omega=(x, y): 12 \le x \le 13, 12 \le y \le 13\}$. Область $G$ представляет собой квадрат со стороной 1 час.
Событие $C$ — {встреча лиц А и В произойдет}. Для встречи необходимо: $|x - y| \le 1/3$ ($20 \text{ мин} = 1/3 \text{ часа}$).
Соответствующая область $g$ в модельном эксперименте примет вид:
$g = \{(x, y) \in G: |x - y| \le 1/3\}$
Данное неравенство распадается на систему:
$y \le x + 1/3$
$y \ge x - 1/3$
Площадь всего квадрата $G$: $mes(G) = S_{G} = 1 \times 1 = 1$.
Площадь заштрихованной области $g$ (весь квадрат минус два пустых угловых треугольника):


$$mes(g) = S_{G} - 2S_{\Delta} = 1 - 2 \left(\frac{1}{2} \times \frac{2}{3} \times \frac{2}{3}\right) = 1 - \frac{4}{9} = \frac{5}{9}$$

$$P(C) = \frac{mes(g)}{mes(G)} = \frac{5/9}{1} = \frac{5}{9}$$


Вероятность того, что встреча состоится, равна $5/9$.

### 4.3. Эмпирический (статистический) подход вычисления вероятности случайных событий

На практике часто встречается случай, когда невозможно применить ни классический, ни геометрический способы.
Например, при стрельбе в мишень мы имеем несчётное множество событий, но попадание в различные области **не равновозможно** (зависит от площади, положения, квалификации стрелка).

В таких случаях вероятность определяют опытным путём — используют **статистический подход**. В его основе лежит стабильность **относительной частоты** $\mu(A, N)/N$ события $A$ при большом числе $N$ опытов.

**Определение 3.** **Статистической вероятностью** события $A$ называется отношение:


$$P^{*}(A, N) = P^{*}(A) = \frac{\mu(A, N)}{N}$$

Свойства статистической вероятности:

1. $P^{*}(\Omega) = N/N = 1$;
2. $P^{*}(\emptyset) = 0/N = 0$;
Из 1) и 2) следует $0 \le P^{*}(A) \le 1$.
3. $P^{*}(A \cup B) = P^{*}(A) + P^{*}(B)$, если только события $A$ и $B$ являются несовместными.

При небольшом числе испытаний статистическая вероятность $P^{*}(A)$ носит нерегулярный характер. Но по мере увеличения числа $N$ она стабилизируется около некоторого постоянного числа $P(A)$.
На содержательном уровне вероятность случайного события $A$ — это число $P(A)$, около которого группируются частоты этого события.

Основоположниками этого подхода являются Ричард фон Мизес и Ганс Рейхенбах. Статистическое понятие вероятности послужило созданию универсального вычислительного метода — **метода Монте-Карло**, который стал основой для метода **имитационного моделирования** реальных процессов.
