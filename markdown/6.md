# Лекция 6. Унифицированная и локализованная вероятностные модели условных экспериментов. Независимость случайных событий

### 6.1. Унифицированная вероятностная модель

**Определение 1.** **Условной (апостериорной, послеопытной) вероятностью** осуществления события $A \in \mathcal{F}$ при наблюдении события $B$, для которого $P(B) \ne 0$, называется число:


$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$


**Важно!** В качестве меры статистической или вероятностной зависимости случайного события $A$ от случайного события $B$ принимают модуль разности $|P(A) - P(A|B)|$.

**Теорема 1.** Упорядоченная тройка $(\Omega, \mathcal{F}, P(\cdot | B))$ является **унифицированной вероятностной моделью** для условного эксперимента $E_y$.
*Доказательство.* Так как $(\Omega, \mathcal{F}, P(\cdot))$ является вероятностной моделью априорного эксперимента $E$, то упорядоченная пара $(\Omega, \mathcal{F})$ обладает всеми необходимыми свойствами в аксиоматике Колмогорова.
Осталось проверить свойства нормировки, неотрицательности и счётной аддитивности (аксиомы Колмогорова) для условной вероятности $P(A|B)$, $A \in \mathcal{F}$:

1. **Аксиома нормировки** для условной вероятности следует из равенств:
$P(\Omega|B) = P(\Omega \cap B) / P(B) = P(B) / P(B) = 1$.
2. **Аксиома неотрицательности** следует из соотношений:
$P(A|B) = P(A \cap B) / P(B) \ge 0$,
так как $P(A \cap B) \ge 0$ и $P(B) > 0$.
3. **Аксиома счетной аддитивности.** Возьмем любую счётную последовательность $A_1, A_2, ...$ попарно непересекающихся событий из $\mathcal{F}$. Для последовательности $A_1, A_2, ...$ находим:

$$P\left(\bigcup_{i=1}^{\infty}A_{i} \Big| B\right) = \frac{P\left(\left(\bigcup_{i=1}^{\infty}A_{i}\right) \cap B\right)}{P(B)} = \frac{P\left(\bigcup_{i=1}^{\infty}(A_{i} \cap B)\right)}{P(B)} = \sum_{i=1}^{\infty}\frac{P(A_{i} \cap B)}{P(B)} = \sum_{i=1}^{\infty}P(A_{i}|B)$$



Здесь пересечение $A_i \cap B \subset A_i$ для всех $i=1, 2, ...$, значит, последовательность $A_1 \cap B, A_2 \cap B, ...$ составлена из попарно непересекающихся случайных событий.

Таким образом можно утверждать, что условная вероятность $P(\cdot | B)$ обладает всеми свойствами априорной вероятности $P(\cdot)$, то есть является неотрицательной, нормированной и счётно-аддитивной функцией на $\sigma$-алгебре $\mathcal{F}$.

### 6.2. Локализованная вероятностная модель

Переходим к построению локализованной вероятностной модели условного эксперимента $E_y$.
Любой элементарный исход из $\Omega$, который не пересекается с $B \in \mathcal{F}$, не происходит при проведении условного эксперимента $E_y$. Исход $B$ всегда наступает при проведении эксперимента $E_y$, поэтому его можно рассматривать как достоверное событие $\Omega_y$ для эксперимента $E_y$.
Все случайные события из $\sigma$-алгебры $\mathcal{F}$, которые не пересекаются с $B$, также не происходят при проведении эксперимента $E_y$ и поэтому могут быть отнесены к ненаблюдаемым событиям по отношению к эксперименту $E_y$.

Таким образом, в качестве $\sigma$-алгебры $\mathcal{F}_y$ наблюдаемых исходов условного эксперимента $E_y$ можно выбрать совокупность вида $\mathcal{F} \cap B$, где $\mathcal{F}$ — $\sigma$-алгебра априорного эксперимента $E$. Доказано, что $\mathcal{F} \cap B$ является $\sigma$-алгеброй подмножеств из $B$.
В случае построения модели $(B, \mathcal{F} \cap B, P_y(A))$, вероятность $P_y(A)$ для $A \in \mathcal{F} \cap B$ может быть найдена и с использованием различных подходов к определению вероятности.

**Определение 2.** Вероятностную модель $(B, \mathcal{F} \cap B, P_y(A))$ условного эксперимента $E_y$ будем называть **локализованной** по отношению к событию $B$.
Так как здесь исход $B$ эксперимента $E$ считаем одним из условий проведения эксперимента $E_y$.

### 6.3. Пример построения условных вероятностных моделей

В урне находятся два белых шара и один чёрный шар. Два лица вынимают из урны одновременно по одному шару. Найти вероятность появления белого шара у первого лица, если у второго появился белый шар.

**Первый способ.** Построим вероятностную модель $(\Omega, \mathcal{F}, P(\cdot))$ априорного эксперимента $E$.
Комплекс условий его проведения $\Sigma = \{u_1$ — взята некоторая урна с тремя шарами, $u_2$ — два белых шара в урне, $u_3$ — один чёрный шар в урне, $u_4$ — два лица одновременно наудачу вынимают по одному шару, ... $\}$. В комплекс условий проведения эксперимента $E$ не входит утверждение о том, что второе лицо изъяло из урны белый шар.

Выберем элементарные случайные события и их описания. Чтобы обеспечить симметрию эксперимента, занумеруем шары в урне. Пусть шары с номерами 1, 2 — белые, а 3 — черный. Номера имеют равные шансы появиться.
$\omega=(x,y)$ — описание элементарного исхода, при котором первое лицо вытаскивает шар с номером $x$, а второе — шар с номером $y$.
Тогда множество $\Omega = \{\omega=(x,y) : x, y \in \{1, 2, 3\}, x \ne y\} = \{\omega_{1}=(1,2), \omega_{2}=(1,3), \omega_{3}=(2,1), \omega_{4}=(2,3), \omega_{5}=(3,1), \omega_{6}=(3,2)\}$.
В $\Omega$ содержится конечное число (шесть) равновозможных элементарных случайных событий. Сигма-алгебра $\mathcal{F}$ — множество всех подмножеств $\Omega$.

Для вычисления значений вероятностной функции $P(\cdot)$ для событий из сигма-алгебры $\mathcal{F}$ будем использовать классическое определение вероятности: $P(A) = \frac{m}{n} = \frac{|A|}{|\Omega|}$.
Например, вероятность для события $D=\{\omega_{1}=(1,2)\}$ и $F=\{\omega_{2}=(1,3), \omega_{4}=(2,3), \omega_{5}=(3,1), \omega_{6}=(3,2)\}$:
$P(D) = \frac{|D|}{|\Omega|} = \frac{1}{6}$,
$P(F) = \frac{|F|}{|\Omega|} = \frac{4}{6}$.
Итак, построена вероятностная модель $(\Omega, \mathcal{F}, P(\cdot))$ для этого априорного эксперимента $E$.

Для нахождения вероятности появления белого шара у первого лица, если у второго появился белый шар, рассмотрим условный эксперимент $E_y$ с комплексом условий $\Sigma_y$.
$\Sigma_y = \{u_1$ — взята некоторая урна с тремя шарами, $u_2$ — два белых шара в урне, $u_3$ — один чёрный шар в урне, $u_4$ — два лица одновременно наудачу вынимают по одному шару, $u_5$ — второе лицо вытаскивает белый шар, ... $\}$.
Построим для $E_y$ унифицированную вероятностную модель $(\Omega, \mathcal{F}, P(\cdot | B))$ на основе вероятностной модели априорного эксперимента.
Рассмотрим следующие события:
$A = \{\omega_{1}=(1,2), \omega_{2}=(1,3), \omega_{3}=(2,1), \omega_{4}=(2,3)\}$ — первое лицо вытаскивает белый шар.
$B = \{\omega_{1}=(1,2), \omega_{3}=(2,1), \omega_{5}=(3,1), \omega_{6}=(3,2)\}$ — второе лицо вытаскивает белый шар.
Пересечение событий $A \cap B = \{\omega_{1}=(1,2), \omega_{3}=(2,1)\}$.
По классическому определению вероятности находим:
$P(B) = \frac{|B|}{|\Omega|} = \frac{4}{6} = \frac{2}{3}$;
$P(A \cap B) = \frac{|A \cap B|}{|\Omega|} = \frac{2}{6} = \frac{1}{3}$.
Теперь по формуле для условной вероятности находим:


$$P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{1}{3} \div \frac{2}{3} = \frac{1}{3} \times \frac{3}{2} = \frac{1}{2}$$

**Второй способ.** Построим для условного эксперимента $E_y$ с комплексом условий $\Sigma_y$ локализованную вероятностную модель $(B, \mathcal{F} \cap B, P_y(\cdot))$.
$B = \{\omega_{1}=(1,2), \omega_{3}=(2,1), \omega_{5}=(3,1), \omega_{6}=(3,2)\}$, $\mathcal{F}_y = \mathcal{F} \cap B$.
Теперь в самом событии $A \cap B = \{\omega_{1}=(1,2), \omega_{3}=(2,1)\} \in \mathcal{F}_y$ уже учтено, что событие $B$ произошло.
Таким образом, нам нужно просто найти для события $A \cap B$ вероятность по классической формуле:


$$P_y(A \cap B) = P(A \cap B) = \frac{|A \cap B|}{|B|} = \frac{2}{4} = \frac{1}{2}$$

Результаты, полученные при обоих способах вычисления, совпадают. То есть при решении можно использовать как унифицированную, так и локализованную вероятностную модель, результат не зависит от вида модели.

### 6.4. Понятие независимости на содержательном уровне

Общеязыковой смысл причинной независимости исходов $A$ и $B$ реального эксперимента $E$ означает, что осуществление события $B$ не влияет на возможность наступления события $A$ или $\overline{A}$.
На опыте проверено, что при большом числе повторов $N$ эксперимента $E$ для независимых событий $A$ и $B$ справедливо приближённое равенство для их относительных частот:


$$\mu(A \cap B, N)/N \approx (\mu(A, N)/N) \times (\mu(B, N)/N)$$


Учитывая определение статистической вероятности $P^{*}(A) = \mu(A, N)/N$, отсюда получается основное соотношение для эмпирической независимости событий $A$ и $B$:


$$P^{*}(A \cap B) \approx P^{*}(A) \times P^{*}(B)$$

### 6.5. Математическое описание независимости случайных событий

**Определение 4.** События $A, B$ из $\mathcal{F}$ называются **статистически независимыми** относительно вероятностной меры $P(\cdot)$, если выполняется следующее условие:


$$P(A \cap B) = P(A)P(B)$$

Используя определение 2, можно доказать, что из независимости событий $A$ и $B$ следует независимость $\overline{A}$ и $B$, $A$ и $\overline{B}$, а также $\overline{A}$ и $\overline{B}$.
Докажем, что из независимости событий $A$ и $B$ следует независимость $\overline{A}$ и $B$:
Так как $A \cup \overline{A} = \Omega$ и $(A \cap \overline{A}) = \emptyset$, то
$P(B) = P(B \cap \Omega) = P(B \cap (A \cup \overline{A})) = P((B \cap A) \cup (B \cap \overline{A})) = P(B \cap A) + P(B \cap \overline{A})$.
Поскольку $P(B \cap A) = P(A)P(B)$, то $P(B) = P(A)P(B) + P(B \cap \overline{A})$,
отсюда $P(B \cap \overline{A}) = P(B) - P(A)P(B) = P(B)(1 - P(A)) = P(B)P(\overline{A})$.
Следовательно, события $\overline{A}$ и $B$ являются независимыми.

Применив формулу для условной вероятности при $P(A_1) > 0$, $P(A_2) > 0$ и статистической независимости событий $A_1$ и $A_2$, получаем:
$P(A_{1}|A_{2}) = \frac{P(A_{1} \cap A_{2})}{P(A_{2})} = \frac{P(A_{1}) \times P(A_{2})}{P(A_{2})} = P(A_{1})$,
$P(A_{2}|A_{1}) = \frac{P(A_{1} \cap A_{2})}{P(A_{1})} = \frac{P(A_{1}) \times P(A_{2})}{P(A_{1})} = P(A_{2})$.
**Важно!** В случае независимости случайных событий $A_1$ и $A_2$ их условные и безусловные вероятности совпадают, т.е. осуществление события $A_2$ не влияет на вероятность появления события $A_1$.

Рассмотрим связь между **несовместностью** и **независимостью** событий $A$ и $B$.
Пусть события $A$ и $B$ несовместимы и $P(B) \ne 0$. Тогда имеем $A \cap B = \emptyset$, $P(A \cap B) = 0$, значит $P(A|B) = P(A \cap B)/P(B) = 0$.
Если $P(A) \ne 0$, то $P(A|B) = 0 \ne P(A)$ и события $A$ и $B$ **зависимы**, а если $P(A) = 0$, то $P(A|B) = 0 = P(A)$ и $A$ и $B$ **независимы**.

**Определение 5.** События $A_1, A_2, ..., A_n$ называются **независимыми в совокупности**, если для любого набора $A_{j1}, A_{j2}, ..., A_{js}$ событий из семейства $\{A_1, A_2, ..., A_n\}$ выполняется равенство:


$$P(A_{j1} \cap A_{j2} \cap ... \cap A_{js}) = P(A_{j1}) \times P(A_{j2}) \times ... \times P(A_{js})$$


где $j_1 < j_2 < ... < j_s$, $1 < s \le n$.
Из попарной независимости событий не следует их независимость в совокупности. Для независимости в совокупности событий недостаточно выполнения лишь равенства $P(A_1 \cap A_2 \cap ... \cap A_n) = P(A_1) \times P(A_2) \times ... \times P(A_n)$.

**Пример 2.** На поверхность стола наудачу подбрасывается симметричная игральная кость два раза и определяется число выпавших очков при каждом броске. Проверить на независимость в совокупности события $A, B$ и $C$.
$A$ — {при втором броске выпадает не более трёх очков}.
$B$ — {при втором броске выпадает не менее трёх и одновременно не более пяти очков}.
$C$ — {сумма выпавших очков при двух бросках равна девяти}.

*Решение.* Обозначим через вектор $\omega=(x,y)$ описание элементарного исхода, когда при первом броске выпадает $x$ очков, а при втором $y$. Здесь имеется всего $n=36$ равновозможных векторов.
Множество $\Omega = \{\omega=(x,y) : x, y = 1, 2, ..., 6\}$.
Через описания элементарных исходов $A, B$ и $C$ представляются:
$A = \{\omega=(x,y) : x = \overline{1,6}, y = 1, 2, 3\} = \{(1,1), ..., (6,3)\}$ (18 исходов).
$B = \{\omega=(x,y) : x = \overline{1,6}, y = 3, 4, 5\} = \{(1,3), ..., (6,5)\}$ (18 исходов).
$C = \{\omega=(x,y) : x, y = \overline{1,6}, x+y=9\} = \{(6,3), (5,4), (4,5), (3,6)\}$ (4 исхода).
Отсюда:
$P(A) = P(B) = 18/36 = 1/2$, $P(C) = 4/36 = 1/9$.
$P(A \cap B) = P(\{(1,3), (2,3), (3,3), (4,3), (5,3), (6,3)\}) = 1/6 \ne P(A)P(B) = 1/4$.
$P(A \cap C) = P(\{(6,3)\}) = 1/36 \ne P(A)P(C) = 1/18$.
$P(B \cap C) = P(\{(6,3), (5,4), (4,5)\}) = 3/36 = 1/12 \ne P(B)P(C) = 1/18$.
Все пары событий зависимы. В то же время:
$P(A \cap B \cap C) = P(\{(6,3)\}) = 1/36 = P(A)P(B)P(C) = (1/2) \times (1/2) \times (1/9) = 1/36$.
Имеет место независимость тройки событий $A, B$ и $C$ (хотя они попарно зависимы).

### 6.6. Независимость относительно различных вероятностных функций

Если теоретико-множественная модель $(\Omega, \mathcal{F})$ фиксирована, то некоторые события $A$ и $B$ могут быть независимыми относительно одной вероятностной функции $P(\cdot)$ и зависимыми относительно другой функции $P^{\prime}(\cdot)$.

**Пример 3.** Симметричная кость. $A$ — {появление чётной грани}, $B$ — {появление грани с цифрой от 3 до 6}.
$P(A) = 1/2$, $P(B) = 2/3$, $P(A \cap B) = 1/3 = P(A)P(B)$. События независимы относительно меры $P(\cdot)$.

Пусть нарушена симметрия:
$P^{\prime}(\{\omega_1\}) = P^{\prime}(\{\omega_3\}) = P^{\prime}(\{\omega_4\}) = P^{\prime}(\{\omega_5\}) = P^{\prime}(\{\omega_6\}) = 4/25$, $P^{\prime}(\{\omega_2\}) = 1/5$.
$P^{\prime}(A) = 13/25$, $P^{\prime}(B) = 16/25$.
$P^{\prime}(A \cap B) = 8/25 \ne P^{\prime}(A)P^{\prime}(B) = (13/25) \times (16/25)$.
События стали зависимыми относительно новой вероятностной меры $P^{\prime}(\cdot)$.

### 6.7. Независимость при изменении условий проведения эксперимента

В следующем примере незначительное изменение условий эксперимента приводит к изменению $(\Omega, \mathcal{F})$ и к потере независимости событий.

**Пример 4.** Пусть имеется стандартная колода из 36 карт. Случайно выбирают одну карту. $A$ = {появление короля}, $B$ = {появление карты пиковой масти}. Проверим на независимость.
$P(A) = 4/36 = 1/9$, $P(B) = 9/36 = 1/4$, $P(A \cap B) = 1/36 = P(A) \times P(B)$.
События независимы.

Добавим в колоду карту «джокер» (теперь 37 карт).
$P(A) = 4/37$, $P(B) = 9/37$, $P(A \cap B) = 1/37 \ne P(A)P(B)$.
События стали зависимыми.
