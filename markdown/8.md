# Лекция 8. Одномерные случайные величины. Основные свойства интегральной функции распределения

### 8.1. Одномерные случайные величины

**Определение 1** (лекция 7). **Одномерной случайной величиной** $\xi(\omega)$ на $(\Omega, \mathcal{F})$ называется всякое отображение множества $\Omega$ на $X \subset \mathbb{R}$, для которого выполняется условие измеримости:


$$\{\omega: \xi(\omega) < x\} \in \mathcal{F} \quad \text{при любом } x \in \mathbb{R}$$

Значения аргумента $\omega$ отображения $\xi(\omega)$ не находятся в нашем распоряжении, а определяются исходом статистически устойчивого эксперимента $E$. Одномерная случайная величина $\xi(\omega): \Omega \to \mathbb{R}$ является математической моделью конкретного измерителя для случайных элементарных событий эксперимента $E$.

**Пример 1.** Пусть $(\Omega, \mathcal{F})$ есть теоретико-множественная модель произвольного статистически устойчивого эксперимента $E$ и $A$ — некоторое случайное событие из $\mathcal{F}$. Рассмотрим функцию $\xi(\omega): \Omega \to \{0, 1\}$, которая задана следующим образом:


$$\xi(\omega) = \begin{cases} 1, & \omega \in A; \\ 0, & \omega \in \overline{A}. \end{cases}$$


Докажем, что функция $\xi(\omega)$ является случайной величиной на $(\Omega, \mathcal{F})$, т.е. для нее выполняется определение 1.
Проверим выполнимость определения 1, для этого построим для функции $\xi(\omega)$ при всех $x \in \mathbb{R}$ множества специального вида $A_x = \{\omega: \xi(\omega) < x\}$:

* при $x \le 0$, $A_x = \{\omega: \xi(\omega) < x\} = \emptyset \in \mathcal{F}$,
* при $0 < x \le 1$, $A_x = \{\omega: \xi(\omega) < x\} = \overline{A} \in \mathcal{F}$,
* при $x > 1$, $A_x = \{\omega: \xi(\omega) < x\} = \Omega \in \mathcal{F}$.

Таким образом, для всех $x \in \mathbb{R}$ множества $A_x = \{\omega: \xi(\omega) < x\}$ принадлежат $\sigma$-алгебре $\mathcal{F}$, определение 1 выполнено и наша функция $\xi(\omega)$ является одномерной случайной величиной на $(\Omega, \mathcal{F})$.

**Пример 2.** Наудачу подбрасывается монета на поверхность стола до первого выпадения герба. Проверить, является ли функция, определяющая число произведённых бросков, случайной величиной в рамках теоретико-множественной модели $(\Omega, \mathcal{F})$ данного эксперимента.

*Решение.* Построим множество всех элементарных случайных событий $\Omega$. Пусть описание $\omega_i$ при каждом $i \ge 1$ соответствует описанию такого элементарного исхода, когда $(i-1)$ раз выпадает решётка, а последний раз выпадает герб. А описание $\omega_{\infty}$ соответствует описанию такого элементарного исхода, когда выпадают только решки и условие останова не срабатывает.
Для рассматриваемого эксперимента множество описаний всех элементарных исходов $\Omega = \{\omega_{\infty}, \omega_1, \omega_2, ...\}$, $\sigma$-алгебра $\mathcal{F} = \{A: A \subset \Omega\}$ — множество всех подмножеств множества $\Omega$.
Пусть функция $\xi(\omega)$ определяет число произведённых бросков в каждом таком опыте. То есть $\xi(\omega_i) = i$ при $i \ge 1$ и $\xi(\omega_{\infty}) = +\infty$.
При любом $x \in \mathbb{R}$ получаем, что $\{\omega: \xi(\omega) < x\} \subset \Omega$ и, значит, $\{\omega: \xi(\omega) < x\} \in \mathcal{F}$.
Но несмотря на это, функция $\xi(\omega)$ **не будет** являться случайной величиной, так как ее значение $\xi(\omega_{\infty}) \notin \mathbb{R}$. Более того, рассматриваемое отображение имеет вид: $\xi(\omega): \Omega \to \{0, 1, ...\} \cup \{+\infty\}$ и таким образом не попадает под определение 1.

### 8.2. Способы задания одномерных случайных величин

**Первый способ.** Поточечное задание случайных величин.
Данный способ заключается в указании для любого $\omega \in \Omega$ численного значения рассматриваемой для этого эксперимента случайной величины, например, $\xi(\omega) = x \in \mathbb{R}$.
Так как элементарные события $\omega$ появляются или не появляются согласно вероятностной функции $P(\cdot): \mathcal{F} \to [0,1]$, то наблюдаемые значения случайной величины $\xi(\omega)$ меняются от опыта к опыту случайным образом.
Таким образом, для полной характеристики свойств случайной величины недостаточно задать только множество $\{\xi(\omega): \omega \in \Omega\} = X$ всех значений $\xi(\omega)$. Необходимо дополнительно указать, с какой вероятностью следует ожидать в каждом отдельном опыте появление того или иного значения случайной величины.

**Второй способ.** Задание одномерной случайной величины $\xi(\omega)$ с помощью функции $F(x) = P(\{\omega: \xi(\omega) < x\}): \mathbb{R} \to [0,1]$.
Функция $F(x) = P(\{\omega: \xi(\omega) < x\})$, $x \in \mathbb{R}$ называется **интегральной функцией распределения** или **интегральным законом распределения** случайной величины $\xi(\cdot)$.
$F(x)$ устанавливает соответствие между возможными значениями случайной величины и вероятностями некоторых событий из $\mathcal{F}$, связанных определенным образом с этими возможными значениями.
Таким образом, вероятностные свойства случайной величины $\xi(\omega)$ полностью определяются её интегральной функцией распределения $F(x)$.
Для краткости вместо $F(x) = P(\{\omega: \xi(\omega) < x\})$ можем писать $F(x) = P(\xi < x)$.
Если рассматриваем в контексте несколько случайных величин, например, $\xi(\cdot), \eta(\cdot)$, то будем обозначать их интегральные функции распределения соответственно через $F_{\xi}(x), F_{\eta}(x), ...$

Поточечное задание случайной величины $\xi(\omega)$ полностью определяет ее интегральную функцию $F(x)$, $x \in \mathbb{R}$, **обратное не верно**. Восстановить случайную величину $\xi(\omega): \Omega \to \mathbb{R}$ по её интегральной функции распределения в общем случае нельзя. Так как различные случайные величины могут определять одну и ту же функцию $F(x)$. Задание случайной величины с помощью интегральной функции распределения является более бедным в информативном смысле, чем её поточечное задание.

**Третий способ.** Задание с помощью операций над другими случайными величинами.
Со случайными величинами $\xi(\omega), \eta(\omega), \xi_1(\omega), \xi_2(\omega)$ произвольного эксперимента $E$ можно проводить различные арифметические действия и операции предельного перехода, если только все эти действия и операции при любом $\omega \in \Omega$ определены в $\mathbb{R}$. В результате этих действий и операций мы получаем новые случайные величины.

Покажем последнее свойство для операций вида $c\xi(\omega)$, где $c = const$, a $\xi(\omega)$ — случайная величина.
Так как $c \in \mathbb{R}$, $\xi(\omega) \in \mathbb{R}$ при любом $\omega \in \Omega$, то $c\xi(\omega): \Omega \to \mathbb{R}$.
Далее проверяем свойство измеримости, используя соотношения (1):

* при $c = 0, x \le 0$ множество $A_x = \{\omega: c\xi(\omega) < x\} = \emptyset \in \mathcal{F}$,
* при $c = 0, x > 0$ множество $A_x = \{\omega: c\xi(\omega) < x\} = \Omega \in \mathcal{F}$.

Пусть теперь $c \ne 0$, тогда:

* при $c > 0$ множество $A_x = \{\omega: c\xi(\omega) < x\} = \{\omega: \xi(\omega) < x/c\} \in \mathcal{F}$,
* при $c < 0$ множество $A_x = \{\omega: c\xi(\omega) < x\} = \{\omega: \xi(\omega) > x/c\} \in \mathcal{F}$.

Таким образом, свойство измеримости выполняется и функция $c\xi(\omega)$ является случайной величиной.

### 8.3. Основные свойства интегральной функции распределения одномерной случайной величины

**Теорема 1.** Интегральная функция распределения $F(x) = P(\{\omega: \xi(\omega) < x\})$ определена при $-\infty \le x \le +\infty$ и принимает значение из отрезка $[0, 1]$.
*Доказательство* теоремы 1 очевидно и следует из определения интегральной функции и простейших свойств вероятностной функции $P(\{\omega: \xi(\omega) < x\})$.

**Теорема 2.** Интегральная функция распределения $F(x)$ является **неубывающей функцией** своего аргумента, то есть из $a_1 < a_2$ следует, что $F(a_1) \le F(a_2)$.
*Доказательство.* Рассмотрим действительные числа $a_1 < a_2$, определим три события:
$A_{a_1} = \{\omega: \xi(\omega) < a_1\}$,
$A_{a_2} = \{\omega: \xi(\omega) < a_2\}$,
$A_{a_1, a_2} = \{\omega: a_1 \le \xi(\omega) < a_2\}$.
По структуре событий имеем $A_{a_2} = A_{a_1} \cup A_{a_1, a_2}$ и $A_{a_1} \cap A_{a_1, a_2} = \emptyset$.
Из равенства и несовместности, на основании аксиомы счётной аддитивности, следует равенство:
$P(A_{a_2}) = P(A_{a_1}) + P(A_{a_1, a_2})$ или $F(a_2) = F(a_1) + P(A_{a_1, a_2})$.
Так как $P(A_{a_1, a_2}) \ge 0$, то $P(A_{a_2}) \ge P(A_{a_1})$ или $F(a_2) \ge F(a_1)$.

Из доказательства теоремы 2 в том числе получаем следующее соотношение:


$$P(A_{a_1, a_2}) = P(\{\omega: a_1 \le \xi(\omega) < a_2\}) = F(a_2) - F(a_1)$$


Случайная величина примет значение из промежутка вида $[a_1, a_2)$ с вероятностью, которая равна разности значений её интегральной функции в точках $a_2$ и $a_1$.

**Теорема 3.** Интегральная функция распределения $F(x)$ **непрерывна слева** при $x \in \mathbb{R}$, то есть $F(a) = F(a-0)$, где $a$ — конечное число $a \in \mathbb{R}$.
*Доказательство.* Рассмотрим $F(a)$, где $a \in \mathbb{R}$. Выберем некоторую числовую последовательность $\{a_n; n \ge 1\}$ такую, что $a_1 < a_2 < ... < a_n < a_{n+1} < ... < a$ и $\lim_{n \to \infty} a_n = a$.
Для доказательства равенства $F(a-0) = \lim_{n \to \infty} F(a_n) = F(a)$ введём последовательность из событий: $A_1 = \{\omega: \xi(\omega) < a_1\}$, $A_2 = \{\omega: \xi(\omega) < a_2\}, ..., A_n = \{\omega: \xi(\omega) < a_n\}, ...$
Заметим, что $A = \{\omega: \xi(\omega) < a\} = \bigcup_{n=1}^{\infty} \{\omega: \xi(\omega) < a_n\}$. Из определения последовательностей следует, что $A_1 \subset A_2 \subset ...$. Поэтому $\lim_{n \to \infty} A_n = \bigcup_{n=1}^{\infty} A_n$. Используя определение $F(a)$ и непрерывность сверху вероятностной функции (вторую аксиому непрерывности), находим:
$F(a) = P(\{\omega: \xi(\omega) < a\}) = P\left(\bigcup_{n=1}^{\infty} \{\omega: \xi(\omega) < a_n\}\right) = P\left(\lim_{n \to \infty} \{\omega: \xi(\omega) < a_n\}\right) = \lim_{n \to \infty} P(\{\omega: \xi(\omega) < a_n\}) = \lim_{n \to \infty} F(a_n) = F(a-0)$.

**Теорема 4.** Вероятность $P(\{\omega: \xi(\omega) \le a\}) = F(a+0)$, $a \in \mathbb{R}$.
*Доказательство.* Рассмотрим $F(a)$, где $a \in \mathbb{R}$, и последовательность $\{a_n; n=1,2,...\}$ такую, что $a_1 > a_2 > ... > a_n > a_{n+1} > ... > a$ и $\lim_{n \to \infty} a_n = a$.
Последовательность порождает невозрастающую последовательность событий $A_1 \supset A_2 \supset ...$
Далее по свойству непрерывности вероятностной функции снизу (первая аксиома непрерывности) и в силу равенства $\{\omega: \xi(\omega) \le a\} = \bigcap_{n=1}^{\infty} \{\omega: \xi(\omega) < a_n\}$ получаем соотношение:
$P(\{\omega: \xi(\omega) \le a\}) = P\left(\bigcap_{n=1}^{\infty} \{\omega: \xi(\omega) < a_n\}\right) = P\left(\lim_{n \to \infty} \{\omega: \xi(\omega) < a_n\}\right) = \lim_{n \to \infty} P(\{\omega: \xi(\omega) < a_n\}) = \lim_{n \to \infty} F(a_n) = F(a+0)$.

На основании соотношений и теоремы 4 получаем формулы:
$P(\{\omega: \xi(\omega) = x\}) = P(\{\omega: \xi(\omega) \le x\}) - P(\{\omega: \xi(\omega) < x\}) = F(x+0) - F(x)$
$P(\{\omega: a \le \xi(\omega) \le b\}) = P(\{\omega: a \le \xi(\omega) < b\}) + P(\{\omega: \xi(\omega) = b\}) = F(b) - F(a) + F(b+0) - F(b) = F(b+0) - F(a)$
$P(\{\omega: \xi(\omega) \ge x\}) = 1 - P(\{\omega: \xi(\omega) < x\}) = 1 - F(x)$
$P(\{\omega: \xi(\omega) > x\}) = 1 - P(\{\omega: \xi(\omega) \le x\}) = 1 - F(x+0)$
$P(\{\omega: a < \xi(\omega) < b\}) = P(\{\omega: a \le \xi(\omega) < b\}) - P(\{\omega: \xi(\omega) = a\}) = F(b) - F(a) - F(a+0) + F(a) = F(b) - F(a+0)$

Итак, вероятность того, что случайная величина примет значение из некоторого стандартного промежутка, определяется её интегральной функцией.
Ранее, в теоремах 3 и 4 мы предполагали, что $a$ — конечное число. Отдельно следует рассмотреть случаи, когда $a = -\infty$ и $a = +\infty$.

**Теорема 5.** Для числовых последовательностей $\{a_n; n \ge 0\}$ и $\{a_{-n}; n \ge 1\}$ таких, что $a_0 < a_1 < ... < a_{n-1} < a_n < ...$, $\lim_{n \to \infty} a_n = +\infty$ и $... < a_{-n} < a_{-n+1} < ... < a_{-1} < a_0$, $\lim_{n \to \infty} a_{-n} = -\infty$, интегральная функция распределения $F(x)$ удовлетворяет **предельным свойствам**:


$$\lim_{x \to +\infty} F(x) = F(+\infty) = 1 \quad \text{и} \quad \lim_{x \to -\infty} F(x) = F(-\infty) = 0$$

**Лемма 1.** Множество значений $x$, в каждом из которых интегральная функция $F(x)$ терпит скачок, или является пустым, или счётным.

Подводя итог, можно сказать, что интегральная функция распределения $F(x)$ любой случайной величины удовлетворяет следующим четырем свойствам:

1. является неубывающей функцией своего аргумента,
2. является непрерывной слева функцией,
3. удовлетворяет неравенствам $0 \le F(x) \le 1$,
4. удовлетворяет предельным равенствам $F(+\infty) = 1$ и $F(-\infty) = 0$.
